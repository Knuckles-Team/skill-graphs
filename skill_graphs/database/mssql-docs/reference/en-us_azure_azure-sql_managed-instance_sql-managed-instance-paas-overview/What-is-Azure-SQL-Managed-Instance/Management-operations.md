## Management operations
Azure SQL Managed Instance provides management operations that you can use to automatically deploy new SQL managed instances, update instance properties, and delete instances when no longer needed. For more information, see [management operations](https://learn.microsoft.com/en-us/azure/azure-sql/managed-instance/management-operations-overview?view=azuresql).
