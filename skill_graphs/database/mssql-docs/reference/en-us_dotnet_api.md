The [Microsoft.SqlServer.Dts.Tasks.XMLTask](https://learn.microsoft.com/en-us/dotnet/api/microsoft.sqlserver.dts.tasks.xmltask?view=sqlserver-2017) namespace contains the interfaces and classes used for the XML Task, which is used to work with XML data. Using this task, a package can retrieve XML documents stored in files, apply operations to the documents using XSL Transformations, (XSLT) style sheets, and XPath expressions, merge multiple documents, and validate, compare, and save the updated documents to files and variables.
