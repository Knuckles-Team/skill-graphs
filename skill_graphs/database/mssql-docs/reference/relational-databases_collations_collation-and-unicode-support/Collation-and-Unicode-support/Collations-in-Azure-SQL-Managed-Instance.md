## Collations in Azure SQL Managed Instance
Server-level collation in Azure SQL Managed Instance can be specified when the instance is created and can't be changed later.
For more information, see [Set or Change the Server Collation](https://learn.microsoft.com/en-us/sql/relational-databases/collations/set-or-change-the-server-collation?view=sql-server-ver17#set-the-server-collation-in-azure-sql-managed-instance).
