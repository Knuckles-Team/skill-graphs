# Tools, Migrators, and Connectors[](https://www.mongodb.com/docs/tools-and-connectors/#tools--migrators--and-connectors "Permalink to this heading")
Find useful tools and integrations to aid in both development and management, including simplified database management, integration, migration, and data visualization.
## CLIs and Utilities[](https://www.mongodb.com/docs/tools-and-connectors/#clis-and-utilities "Permalink to this heading")
![Atlas CLI LogoMark](https://webimages.mongodb.com/_com_assets/icons/atlas_cli.svg)
Atlas CLI
Learn how to use the Atlas CLI to manage your MongoDB Atlas deployment.
![MongoDB CLI LogoMark](https://webimages.mongodb.com/_com_assets/icons/mdb_cli.svg)
MongoDB CLI
Use the MongoDB CLI to manage your MongoDB services from the command line.
![Database Tools LogoMark](https://webimages.mongodb.com/_com_assets/icons/general_features_tools.svg)
Database Tools
Export data from one MongoDB cluster to a JSON, BSON, CSV, or TSV file and import it to another MongoDB cluster.
![MongoDB Shell LogoMark](https://webimages.mongodb.com/_com_assets/icons/mdb_shell.svg)
Mongo Shell
Learn about the `mongosh` tool to test queries and interact with your MongoDB database.
## Connectors[](https://www.mongodb.com/docs/tools-and-connectors/#connectors "Permalink to this heading")
![SQL Connector LogoMark](https://webimages.mongodb.com/_com_assets/icons/connectors_sql_connector.svg)
SQL Interface
Simplify exporting your MongoDB data to SQL-driven tools with native driver support for all deployment types.
![BI Connector LogoMark](https://webimages.mongodb.com/_com_assets/icons/connectors_bi_connector.svg)
BI Connector
Connect MongoDB to your favorite BI platforms to visualize, graph, and report on your MongoDB data.
![Kafka Connector LogoMark](https://webimages.mongodb.com/_com_assets/icons/connectors_kafka.svg)
Kafka Connector
Build robust pipelines that stream events between applications with MongoDB and Apache Kafka.
![Spark LogoMark](https://webimages.mongodb.com/_com_assets/icons/connectors.svg)
Spark Connector
Leverage Apache Spark's libraries to analyze MongoDB data with machine learning, graph, and SQL APIs.
## IDE and AI Integrations[](https://www.mongodb.com/docs/tools-and-connectors/#ide-and-ai-integrations "Permalink to this heading")
![MCP server icon](https://webimages.mongodb.com/_com_assets/icons/general_features_tools.svg)
MongoDB MCP Server
Interact with your MongoDB data and deployments from your agentic AI tools, assistants, and platforms.
![C# Analyzer LogoMark](https://webimages.mongodb.com/_com_assets/icons/general_features_build_faster.svg)
C# Analyzer
Understand and troubleshoot queries and aggregations directly inside Visual Studio and other tools.
![VS Code Extension LogoMark](https://webimages.mongodb.com/_com_assets/icons/mdb_shell.svg)
VS Code Extension
Find out how you can manage your data directly within your coding environment.
![IntelliJ Plugin LogoMark](https://webimages.mongodb.com/_com_assets/icons/mdb_shell.svg)
IntelliJ Plugin
Build applications and work with your MongoDB data directly from IntelliJ IDEA.
## Migrators[](https://www.mongodb.com/docs/tools-and-connectors/#migrators "Permalink to this heading")
![Cluster-to-Cluster Sync LogoMark](https://webimages.mongodb.com/_com_assets/icons/mdb_cluster_to_cluster_sync.svg)
Mongosync
Continuously synchronize data between MongoDB clusters in the same or hybrid environments, including Atlas, private cloud, on-premises, and at the edge.
![Relational Migrator LogoMark](https://webimages.mongodb.com/_com_assets/icons/mdb_migrator.svg)
Relational Migrator
Migrate data from your relational database to MongoDB.
## Management Tools[](https://www.mongodb.com/docs/tools-and-connectors/#management-tools "Permalink to this heading")
![Cloud Manager LogoMark](https://webimages.mongodb.com/_com_assets/icons/atlas_cloud_manager.svg)
Cloud Manager
Use the hosted management platform to monitor, automate, and backup self-managed deployments.
![Ops Manager LogoMark](https://webimages.mongodb.com/_com_assets/icons/enterpriseadvanced_ops_manager.svg)
Ops Manager
Deploy, monitor, back up, and scale MongoDB on your own infrastructure.
## Partner Integrations[](https://www.mongodb.com/docs/tools-and-connectors/#partner-integrations "Permalink to this heading")
![Partner Integrations LogoMark](https://webimages.mongodb.com/_com_assets/icons/atlas_integration.svg)
Overview
Learn how MongoDB integrates with partner technologies to enhance your applications and workflows.
![Render LogoMark](https://webimages.mongodb.com/_com_assets/icons/atlas_integration.svg)
Render
Connect your Render-hosted applications to an Atlas cluster.
![Vercel LogoMark](https://webimages.mongodb.com/_com_assets/icons/atlas_integration.svg)
Vercel
Connect your Atlas clusters to applications that you deploy using Vercel.
![Azure LogoMark](https://webimages.mongodb.com/_com_assets/icons/atlas_integration.svg)
Azure
Learn about the integration between MongoDB Atlas and Microsoft Azure.
## Visualization[](https://www.mongodb.com/docs/tools-and-connectors/#visualization "Permalink to this heading")
