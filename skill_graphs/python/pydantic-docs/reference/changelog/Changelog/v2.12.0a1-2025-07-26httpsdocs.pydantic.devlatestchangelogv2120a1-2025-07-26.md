## v2.12.0a1 (2025-07-26)[¶](https://docs.pydantic.dev/latest/changelog/#v2120a1-2025-07-26)
This is the first alpha release of the upcoming 2.12 release, which adds initial support for Python 3.14.

  * Add regex patterns to JSON schema for `Decimal` type by
  * Add support for `doc` attribute on dataclass fields by
