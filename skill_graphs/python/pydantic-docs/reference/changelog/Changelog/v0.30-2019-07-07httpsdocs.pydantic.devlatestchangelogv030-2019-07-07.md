## v0.30 (2019-07-07)[¶](https://docs.pydantic.dev/latest/changelog/#v030-2019-07-07)
  * enforce single quotes in code,
  * fix infinite recursion with dataclass inheritance and `__post_init__`,
  * fix default values for `GenericModel`,
  * clarify that self-referencing models require Python 3.7+,
  * fix truncate for types,
  * add `alias_generator` support,
  * fix unparameterized generic type schema generation,
  * fix schema generation with multiple/circular references to the same model,
  * support custom root types,
  * support `self` as a field name in `parse_obj`,
