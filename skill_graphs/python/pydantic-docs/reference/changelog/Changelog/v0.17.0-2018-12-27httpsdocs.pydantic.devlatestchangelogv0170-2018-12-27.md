  * fix schema for `timedelta` as number,
  * prevent validators being called repeatedly after inheritance,
  * prevent duplicate validator check in ipython, fix
  * add "Using Pydantic" section to docs,
  * fix schema generation for fields annotated as `: dict`, `: list`, `: tuple` and `: set`,

  * support for nested dataclasses,
  * better errors when getting an `ImportError` with `PyObject`,
  * rename `get_validators` to `__get_validators__`, deprecation warning on use of old name,
  * support `ClassVar` by excluding such attributes from fields,
