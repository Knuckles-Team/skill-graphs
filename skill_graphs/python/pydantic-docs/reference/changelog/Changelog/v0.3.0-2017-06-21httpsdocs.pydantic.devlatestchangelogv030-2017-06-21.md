## v0.3.0 (2017-06-21)[¶](https://docs.pydantic.dev/latest/changelog/#v030-2017-06-21)
  * immutable models via `config.allow_mutation = False`, associated cleanup and performance improvement
  * immutable helper methods `construct()` and `copy()`
  * allow pickling of models
  * `setattr` is removed as `__setattr__` is now intelligent
  * `raise_exception` removed, Models now always raise exceptions
  * instance method validators removed
  * django-restful-framework benchmarks added
  * fix inheritance bug
  * make str type stricter so list, dict etc are not coerced to strings.
  * add `StrictStr` which only always strings as input
