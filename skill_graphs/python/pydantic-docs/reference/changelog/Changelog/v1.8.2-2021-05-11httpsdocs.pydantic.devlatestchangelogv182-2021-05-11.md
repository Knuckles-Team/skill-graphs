## v1.8.2 (2021-05-11)[¶](https://docs.pydantic.dev/latest/changelog/#v182-2021-05-11)
Warning
A security vulnerability, level "moderate" is fixed in v1.8.2. Please upgrade **ASAP**. See security advisory
  * **Security fix:** Fix `date` and `datetime` parsing so passing either `'infinity'` or `float('inf')` (or their negative values) does not cause an infinite loop, see security advisory
  * fix schema generation with Enum by generating a valid name,
  * fix JSON schema generation with a `Literal` of an enum member,
  * Fix bug with configurations declarations that are passed as keyword arguments during class creation,
  * Allow passing `json_encoders` in class kwargs,
  * support arbitrary types with custom `__eq__`,
  * support `Annotated` in `validate_arguments` and in generic models with Python 3.9,
