# SciPy documentation[#](https://docs.scipy.org/doc/scipy-1.10.1/index.html#scipy-documentation "Permalink to this heading")
**Date** : February 19, 2023 **Version** : 1.10.1

**SciPy** (pronounced “Sigh Pie”) is an open-source software for mathematics, science, and engineering.
![card-img-top](https://docs.scipy.org/doc/scipy-1.10.1/_images/index_getting_started.png)
Getting started
New to _SciPy_? Check out the getting started guides. They contain an introduction to _SciPy’s_ main concepts and links to additional tutorials.
[To the getting started guides](https://docs.scipy.org/doc/scipy-1.10.1/getting_started.html#getting-started-ref)
![card-img-top](https://docs.scipy.org/doc/scipy-1.10.1/_images/index_user_guide.png)
User guide
The user guide provides in-depth information on the key concepts of SciPy with useful background information and explanation.
[To the user guide](https://docs.scipy.org/doc/scipy-1.10.1/tutorial/index.html#user-guide)
![card-img-top](https://docs.scipy.org/doc/scipy-1.10.1/_images/index_api.png)
API reference
The reference guide contains a detailed description of the SciPy API. The reference describes how the methods work and which parameters can be used. It assumes that you have an understanding of the key concepts.
[To the reference guide](https://docs.scipy.org/doc/scipy-1.10.1/reference/index.html#scipy-api)

Saw a typo in the documentation? Want to improve existing functionalities? The contributing guidelines will guide you through the process of improving SciPy.
[To the development guide](https://docs.scipy.org/doc/scipy-1.10.1/dev/index.html#scipy-development)
