# Animations using Matplotlib[#](https://matplotlib.org/stable/users/explain/animations/index.html#animations-using-matplotlib "Link to this heading")
Based on its plotting functionality, Matplotlib also provides an interface to generate animations using the [`animation`](https://matplotlib.org/stable/api/animation_api.html#module-matplotlib.animation "matplotlib.animation") module. An animation is a sequence of frames where each frame corresponds to a plot on a [`Figure`](https://matplotlib.org/stable/api/_as_gen/matplotlib.figure.Figure.html#matplotlib.figure.Figure "matplotlib.figure.Figure"). This tutorial covers a general guideline on how to create such animations and the different options available.
