#  `tabnanny` — Detection of ambiguous indentation[¶](https://docs.python.org/3/library/tabnanny.html#module-tabnanny "Link to this heading")
**Source code:**
* * *
For the time being this module is intended to be called as a script. However it is possible to import it into an IDE and use the function [`check()`](https://docs.python.org/3/library/tabnanny.html#tabnanny.check "tabnanny.check") described below.
Note
The API provided by this module is likely to change in future releases; such changes may not be backward compatible.

tabnanny.check(_file_or_dir_)[¶](https://docs.python.org/3/library/tabnanny.html#tabnanny.check "Link to this definition")

If _file_or_dir_ is a directory and not a symbolic link, then recursively descend the directory tree named by _file_or_dir_ , checking all `.py` files along the way. If _file_or_dir_ is an ordinary Python source file, it is checked for whitespace related problems. The diagnostic messages are written to standard output using the [`print()`](https://docs.python.org/3/library/functions.html#print "print") function.

tabnanny.verbose[¶](https://docs.python.org/3/library/tabnanny.html#tabnanny.verbose "Link to this definition")

Flag indicating whether to print verbose messages. This is incremented by the `-v` option if called as a script.

tabnanny.filename_only[¶](https://docs.python.org/3/library/tabnanny.html#tabnanny.filename_only "Link to this definition")

Flag indicating whether to print only the filenames of files containing whitespace related problems. This is set to true by the `-q` option if called as a script.

_exception_ tabnanny.NannyNag[¶](https://docs.python.org/3/library/tabnanny.html#tabnanny.NannyNag "Link to this definition")

Raised by [`process_tokens()`](https://docs.python.org/3/library/tabnanny.html#tabnanny.process_tokens "tabnanny.process_tokens") if detecting an ambiguous indent. Captured and handled in [`check()`](https://docs.python.org/3/library/tabnanny.html#tabnanny.check "tabnanny.check").

tabnanny.process_tokens(_tokens_)[¶](https://docs.python.org/3/library/tabnanny.html#tabnanny.process_tokens "Link to this definition")

This function is used by [`check()`](https://docs.python.org/3/library/tabnanny.html#tabnanny.check "tabnanny.check") to process tokens generated by the [`tokenize`](https://docs.python.org/3/library/tokenize.html#module-tokenize "tokenize: Lexical scanner for Python source code.") module.
See also

Module [`tokenize`](https://docs.python.org/3/library/tokenize.html#module-tokenize "tokenize: Lexical scanner for Python source code.")

Lexical scanner for Python source code.
