### Features[¶](https://fastapi.tiangolo.com/release-notes/#features_36)
  * ✨ Export `WebSocketState` in `fastapi.websockets`. PR
  * ✨ Support Python internal description on Pydantic model's docstring. PR
  * ✨ Update `ORJSONResponse` to support non `str` keys and serializing Numpy arrays. PR


### Fixes[¶](https://fastapi.tiangolo.com/release-notes/#fixes_56)
  * 🐛 Allow exit code for dependencies with `yield` to always execute, by removing capacity limiter for them, to e.g. allow closing DB connections without deadlocks. PR
  * 🐛 Fix FastAPI People GitHub Action: set HTTPX timeout for GraphQL query request. PR
  * 🐛 Make sure a parameter defined as required is kept required in OpenAPI even if defined as optional in another dependency. PR
  * 🐛 Fix support for path parameters in WebSockets. PR
