### Breaking Changes - Upgrade[¶](https://fastapi.tiangolo.com/release-notes/#breaking-changes-upgrade_1)
This release adds support for
It upgrades the version of Starlette to `0.15.0`, now based on **FastAPI** are now based on AnyIO as well, making it compatible with both **asyncio** and **Trio**.
You can read the docs about running [FastAPI with Trio using Hypercorn](https://fastapi.tiangolo.com/deployment/manually/#hypercorn-with-trio).
This release also removes `graphene` as an optional dependency for GraphQL. If you need to work with GraphQL, the recommended library now is [FastAPI with GraphQL docs](https://fastapi.tiangolo.com/advanced/graphql/).
