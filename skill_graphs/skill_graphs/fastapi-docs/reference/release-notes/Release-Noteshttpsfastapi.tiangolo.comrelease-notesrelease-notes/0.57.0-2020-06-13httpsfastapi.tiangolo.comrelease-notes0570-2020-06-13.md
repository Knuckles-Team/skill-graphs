  * Enable `showCommonExtensions` in Swagger UI to show additional validations like `maxLength`, etc. PR

  * Add docs about [Default response class](https://fastapi.tiangolo.com/advanced/custom-response/#default-response-class). PR
  * Add note in docs about additional parameters `response_model_exclude_defaults` and `response_model_exclude_none` in [Response Model](https://fastapi.tiangolo.com/tutorial/response-model/#use-the-response_model_exclude_unset-parameter). PR
