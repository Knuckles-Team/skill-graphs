  * Update docs for [Development - Contributing: Translations](https://fastapi.tiangolo.com/contributing/#translations) including note about reviewing translation PRs.
  * Update log style in README.md for GitHub Markdown compatibility. PR
  * Add Python venv `env` to `.gitignore`. PR
  * Start Portuguese translations. PR
  * Update docs for Pydantic's `Settings` using a dependency with `@lru_cache()`. PR
  * Add first translation to Spanish [FastAPI](https://fastapi.tiangolo.com/es/). PR
  * Add docs about [Settings and Environment Variables](https://fastapi.tiangolo.com/advanced/settings/). Initial PR
