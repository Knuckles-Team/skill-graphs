## 0.75.2 (2022-04-17)[¶](https://fastapi.tiangolo.com/release-notes/#0752-2022-04-17)
This release includes upgrades to third-party packages that handle security issues. Although there's a chance these issues don't affect you in particular, please upgrade as soon as possible.
### Fixes[¶](https://fastapi.tiangolo.com/release-notes/#fixes_60)
  * ✅ Fix new/recent tests with new fixed `ValidationError` JSON Schema. PR
  * 🐛 Fix JSON Schema for `ValidationError` at field `loc`. PR
  * 🐛 Fix support for prefix on APIRouter WebSockets. PR


### Upgrades[¶](https://fastapi.tiangolo.com/release-notes/#upgrades_34)
  * ⬆️ Update ujson ranges for CVE-2021-45958. PR
  * ⬆️ Upgrade dependencies upper range for extras "all". PR
  * ⬆ Upgrade Swagger UI - swagger-ui-dist@4. This handles a security issue in Swagger UI itself where it could be possible to inject HTML into Swagger UI. Please upgrade as soon as you can, in particular if you expose your Swagger UI (`/docs`) publicly to non-expert users. PR


### Internal[¶](https://fastapi.tiangolo.com/release-notes/#internal_106)
  * 🔧 Update sponsors, add: ExoFlare, Ines Course; remove: Dropbase, Vim.so, Calmcode; update: Striveworks, TalkPython and TestDriven.io. PR
  * ⬆️ Upgrade Codecov GitHub Action. PR
