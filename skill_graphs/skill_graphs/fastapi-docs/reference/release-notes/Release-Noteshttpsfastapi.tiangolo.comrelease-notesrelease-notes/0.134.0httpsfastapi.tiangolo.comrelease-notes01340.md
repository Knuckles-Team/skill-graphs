## 0.134.0[¶](https://fastapi.tiangolo.com/release-notes/#01340)
### Features[¶](https://fastapi.tiangolo.com/release-notes/#features_1)
  * ✨ Add support for streaming JSON Lines and binary data with `yield`. PR
    * This also upgrades Starlette from `>=0.40.0` to `>=0.46.0`, as it's needed to properly unrwap and re-raise exceptions from exception groups.
    * New docs: [Stream JSON Lines](https://fastapi.tiangolo.com/tutorial/stream-json-lines/).
    * And new docs: [Stream Data](https://fastapi.tiangolo.com/advanced/stream-data/).


### Docs[¶](https://fastapi.tiangolo.com/release-notes/#docs_2)
  * 📝 Update Library Agent Skill with streaming responses. PR
  * 📝 Update docs for responses and new stream with `yield`. PR
  * 📝 Add `await` in `StreamingResponse` code example to allow cancellation. PR
  * 📝 Rename `docs_src/websockets` to `docs_src/websockets_` to avoid import errors. PR


### Internal[¶](https://fastapi.tiangolo.com/release-notes/#internal_2)
  * 🔨 Run tests with `pytest-xdist` and `pytest-cov`. PR
