## 0.73.0 (2022-01-23)[¶](https://fastapi.tiangolo.com/release-notes/#0730-2022-01-23)
### Features[¶](https://fastapi.tiangolo.com/release-notes/#features_42)
  * ✨ Add support for declaring `UploadFile` parameters without explicit `File()`. PR [Request Files - File Parameters with UploadFile](https://fastapi.tiangolo.com/tutorial/request-files/#file-parameters-with-uploadfile).
  * ✨ Add support for tags with Enums. PR [Path Operation Configuration - Tags with Enums](https://fastapi.tiangolo.com/tutorial/path-operation-configuration/#tags-with-enums).
  * ✨ Allow hiding from OpenAPI (and Swagger UI) `Query`, `Cookie`, `Header`, and `Path` parameters. PR [Query Parameters and String Validations - Exclude from OpenAPI](https://fastapi.tiangolo.com/tutorial/query-params-str-validations/#exclude-from-openapi).
