## 0.116.0 (2025-07-07)[¶](https://fastapi.tiangolo.com/release-notes/#01160-2025-07-07)
### Features[¶](https://fastapi.tiangolo.com/release-notes/#features_12)
  * ✨ Add support for deploying to FastAPI Cloud with `fastapi deploy`. PR


Installing `fastapi[standard]` now includes `fastapi-cloud-cli`.
This will allow you to deploy to `fastapi deploy` command.
If you want to install `fastapi` with the standard dependencies but without `fastapi-cloud-cli`, you can install instead `fastapi[standard-no-fastapi-cloud-cli]`.
