## Capella Analytics SDKs
[Capella Analytics](https://docs.couchbase.com/analytics/intro/intro.html) is an analytical database (RT-OLAP) for real time apps and operational intelligence. [Columnar SDKs](https://docs.couchbase.com/home/columnar-sdk.html) for Capella Analytics are tailored to the APIs offered by this service.
