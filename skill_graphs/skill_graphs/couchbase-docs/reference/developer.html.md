# Develop with Couchbase
The Developer Data Platform for Critical Applications in Our AI World.
Couchbase is a multipurpose NoSQL database for transactional, analytical, mobile, and AI applications. Develop at the edge with offline-first Couchbase Lite, for transactional workloads with SDKs in a dozen popular programming languages, for real-time analytics, and build agentic apps (currently in private preview).
## Development Choices for Your Use Cases
###  Transactional Workloads — in the Cloud or your Datacenter
Develop an app for [Capella Operational](https://docs.couchbase.com/cloud/develop/intro.html), or a [self-managed Couchbase Server](https://docs.couchbase.com/server/7.6/develop/intro.html) — with [Operational SDKs](https://docs.couchbase.com/home/sdk.html). Explore SQL++, or the fast CRUD interface of the Data Service.
  * [Get Started with the Java SDK](https://docs.couchbase.com/java-sdk/current/hello-world/start-using-sdk.html)
  * [NoSQL for RDBMS users](https://docs.couchbase.com/cloud/tutorials/couchbase-tutorial-student-records.html)
  * Try our [other SDKs](https://docs.couchbase.com/home/sdk.html):
[C](https://docs.couchbase.com/c-sdk/current/hello-world/overview.html) | [C++](https://docs.couchbase.com/cxx-sdk/current/hello-world/overview.html) | [.NET](https://docs.couchbase.com/dotnet-sdk/current/hello-world/overview.html) | [Go](https://docs.couchbase.com/go-sdk/current/hello-world/overview.html) | [Kotlin](https://docs.couchbase.com/kotlin-sdk/current/hello-world/overview.html) | [Node.js](https://docs.couchbase.com/nodejs-sdk/current/hello-world/overview.html) | [PHP](https://docs.couchbase.com/php-sdk/current/hello-world/overview.html) | [Python](https://docs.couchbase.com/python-sdk/current/hello-world/overview.html) | [Ruby](https://docs.couchbase.com/ruby-sdk/current/hello-world/overview.html) | [Rust](https://docs.couchbase.com/rust-sdk/current/hello-world/overview.html) | [Scala](https://docs.couchbase.com/scala-sdk/current/hello-world/overview.html)
  * [Integrations, Connectors, and Tools](https://docs.couchbase.com/cloud/third-party/integrations.html)


###  Real Time Analytics with Analytics SDKs
[Enterprise Analytics](https://docs.couchbase.com/enterprise-analytics/current/intro/intro.html) brings the power of NoSQL to the world of analytics. Self-managed Enterprise Analytics brings real-time adaptive applications to your datacenter or private cloud — or try this service fully-hosted as [Capella Analytics](https://docs.couchbase.com/analytics/intro/intro.html), which integrates seamlessly with the Couchbase Capella cloud platform. [Analytics SDKs](https://docs.couchbase.com/home/analytics-sdk.html) support streaming APIs to handle large datasets.
  * [.NET Analytics SDK Docs](https://docs.couchbase.com/dotnet-analytics-sdk/current/hello-world/overview.html)
  * [Go Analytics SDK Docs](https://docs.couchbase.com/go-analytics-sdk/current/hello-world/overview.html)
  * [Java Analytics SDK Docs](https://docs.couchbase.com/java-analytics-sdk/current/hello-world/overview.html)
  * [Node.js Analytics SDK Docs](https://docs.couchbase.com/nodejs-analytics-sdk/current/hello-world/overview.html)
  * [Python Analytics SDK Docs](https://docs.couchbase.com/python-analytics-sdk/current/hello-world/overview.html)


###  Develop for Mobile and the Edge
Build your app with [Couchbase Lite](https://docs.couchbase.com/couchbase-lite/current/index.html) for offline-first connectivity, then sync to Couchbase Server with Capella App services (or self-managed Sync Gateway) — or run peer-to-peer.
  * [Develop for Android in Java or Kotlin](https://docs.couchbase.com/couchbase-lite/current/android/quickstart.html)
  * [Develop for iOS on Swift](https://docs.couchbase.com/couchbase-lite/current/swift/quickstart.html) (and [Objective-C](https://docs.couchbase.com/couchbase-lite/current/objc/quickstart.html))
  * [Develop for Edge devices in C](https://docs.couchbase.com/couchbase-lite/current/c/quickstart.html)
  * Couchbase Lite [on C# .NET](https://docs.couchbase.com/couchbase-lite/current/csharp/quickstart.html) | [Java](https://docs.couchbase.com/couchbase-lite/current/java/quickstart.html) | [JavaScript — Ionic and React](https://docs.couchbase.com/home/developer.html#couchbase-lite:javascript:quickstart.adoc)
  * Sync your data with [App Services](https://docs.couchbase.com/home/developer.html#cloud:app-services:index.adoc) / self-managed [SyncGateway](https://docs.couchbase.com/sync-gateway/current/introduction.html) / [Edge Server](https://docs.couchbase.com/couchbase-edge-server/current/introduction/intro.html) — or run [peer-to-peer](https://docs.couchbase.com/couchbase-lite/current/swift/p2psync-websocket.html).


###  Develop RAG and Agentic AI Applications
Our Vector Search Service facilitates RAG applications — and offers the ability to combine searches with our sophisticated Search API. Agentic Apps can be built with Agent Catalog (currently in Private Preview).
  * [Vector Search](https://docs.couchbase.com/cloud/vector-search/vector-search.html)
  * [Agent Catalog Private Preview](https://info.couchbase.com/capella-ai-services-signup)
