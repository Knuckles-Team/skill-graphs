## [Testing](https://laravel.com/docs/12.x/cashier-paddle#testing)
While testing, you should manually test your billing flow to make sure your integration works as expected.
For automated tests, including those executed within a CI environment, you may use [Laravel's HTTP Client](https://laravel.com/docs/12.x/http-client#testing) to fake HTTP calls made to Paddle. Although this does not test the actual responses from Paddle, it does provide a way to test your application without actually calling Paddle's API.
