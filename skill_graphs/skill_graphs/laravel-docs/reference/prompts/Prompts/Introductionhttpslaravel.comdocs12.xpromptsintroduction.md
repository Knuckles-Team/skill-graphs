## [Introduction](https://laravel.com/docs/12.x/prompts#introduction)
![](https://laravel.com/img/docs/prompts-example.png)
Laravel Prompts is perfect for accepting user input in your [Artisan console commands](https://laravel.com/docs/12.x/artisan#writing-commands), but it may also be used in any command-line PHP project.
Laravel Prompts supports macOS, Linux, and Windows with WSL. For more information, please see our documentation on [unsupported environments & fallbacks](https://laravel.com/docs/12.x/prompts#fallbacks).
