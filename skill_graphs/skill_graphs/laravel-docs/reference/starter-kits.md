###  An authenticated application out of the box
Authentication options to suit your application's needs.
Optional
### Built-in Laravel authentication
optional
Everything you need for a robust and secure authentication. Including, login, registration, password reset, and email verification.
![](https://laravel.com/images/starter-kits/features-auth.png) ![](https://laravel.com/images/starter-kits/features-auth-dark.png)
Optional
### WorkOS AuthKit authentication
optional
Login with GitHub, Google, Microsoft, or Apple ID, Passkey support, code sent via email, built-in rate limiting, geographical protection, and SSO.
![](https://laravel.com/images/starter-kits/features-workos.png) ![](https://laravel.com/images/starter-kits/features-workos-dark.png)
### Customer dashboard
A clean slate to start building your next big idea

Built-in user profile, password, and appearance management
