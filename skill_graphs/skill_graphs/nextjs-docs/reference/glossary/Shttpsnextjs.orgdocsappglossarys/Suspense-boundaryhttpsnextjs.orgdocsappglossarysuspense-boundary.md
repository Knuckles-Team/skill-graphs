## Suspense boundary[](https://nextjs.org/docs/app/glossary#suspense-boundary)
A React [static shell](https://nextjs.org/docs/app/glossary#static-shell) ends and [streaming](https://nextjs.org/docs/app/glossary#streaming) begins, enabling [Partial Prerendering](https://nextjs.org/docs/app/glossary#partial-prerendering-ppr).
