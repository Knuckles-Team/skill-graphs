# How to set up Cypress with Next.js
Last updated February 27, 2026
**End-to-End (E2E)** and **Component Testing**. This page will show you how to set up Cypress with Next.js and write your first tests.
> **Warning:**
>   * Cypress versions below 13.6.3 do not support `moduleResolution:"bundler"`. However, this issue has been resolved in Cypress version 13.6.3 and later.
