# Start Developing with MongoDB[](https://www.mongodb.com/docs/drivers/#start-developing-with-mongodb "Permalink to this heading")
You can connect your application to your MongoDB Atlas deployment or a self-hosted MongoDB cluster by using one of the official MongoDB libraries. To learn more about Atlas, see [What is MongoDB Atlas?](https://www.mongodb.com/docs/atlas/). To learn how to download and install a self-hosted MongoDB cluster, see [Install MongoDB.](https://www.mongodb.com/docs/manual/installation/)
The following libraries are officially supported by MongoDB. MongoDB actively develops new features, adds performance enhancements, fixes bugs, and applies security patches to them.
