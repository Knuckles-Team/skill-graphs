## Como a documentação é organizada[¶](https://docs.djangoproject.com/pt-br/6.0/#how-the-documentation-is-organized "Link para este cabeçalho")
Django tem uma grande quantidade de documentação. Uma visão geral de como ele é organizado vai te ajudar a saber aonde procurar para certas coisas:
  * [Tutoriais](https://docs.djangoproject.com/pt-br/6.0/intro/) conduzem você pela mão através de uma série de passos para criar uma aplicação web. Comece aqui se você é novo no Django ou no desenvolvimento de aplicações web. Veja também o “[Primeiros passos](https://docs.djangoproject.com/pt-br/6.0/#index-first-steps)”.
  * [Guia de tópicos](https://docs.djangoproject.com/pt-br/6.0/topics/) discute temas-chave e conceitos em um nível relativamente alto, fornece explicaçõesde modo de funcionamento e informações úteis.
  * [Guiaa de referências](https://docs.djangoproject.com/pt-br/6.0/ref/) contém referência técnica para APIs e outros aspectos do maquinário do Django. Eles descrevem como isso funciona e como usá-los, mas assume que você tem um entendimento básico dos conceitos chaves.
  * [Guias ‘How-to’](https://docs.djangoproject.com/pt-br/6.0/howto/) são receitas. Eles guiam você através de passos baseados na abordagem de problemas chave e casos de uso. Eles são mais avançados que os tutoriais e assumem que você já tem algum conhecimento de como o Django trabalha.
