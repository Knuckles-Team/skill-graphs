# System check framework[¶](https://docs.djangoproject.com/en/5.0/ref/checks/#system-check-framework "Link to this heading")
The system check framework is a set of static checks for validating Django projects. It detects common problems and provides hints for how to fix them. The framework is extensible so you can easily add your own checks.
For details on how to add your own checks and integrate them with Django’s system checks, see the [System check topic guide](https://docs.djangoproject.com/en/5.0/topics/checks/).
