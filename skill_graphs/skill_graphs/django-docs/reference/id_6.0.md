## Bagaimana dokumentasi diatur[¶](https://docs.djangoproject.com/id/6.0/#how-the-documentation-is-organized "Link to this heading")
Django mempunyai banyak dokumentasi. Ikhstisar tingkat-tinggi dari bagaimana dia disusun akan membantu anda mengetahui dimana mencari untuk hal-hal tertentu:
  * [Tutorials](https://docs.djangoproject.com/id/6.0/intro/) membawamu dengan tangan melalui rangkaian dari langkah untuk membuat aplikasi jaringan. Dimulai disini jika anda baru terhadap Django atau pengembangan aplikasi. Juga lihat "[Langkah awal](https://docs.djangoproject.com/id/6.0/#index-first-steps)".
  * [Topic guides](https://docs.djangoproject.com/id/6.0/topics/) mengobrolkan kunci topik dan konsep pada tingkat tinggi dan menyediakan informasi latar belakang berguna dan penjelasan.
  * [Reference guides](https://docs.djangoproject.com/id/6.0/ref/) mengandung acuan teknis untuk API dan aspek lain dari perlengkapan Django. Mereka menggambarkan bagaimana itu bekerja dan bagaimana menggunakannya bahwa anda mempunyai pemahaman dasar dari kunci konsep.
  * [How-to guides](https://docs.djangoproject.com/id/6.0/howto/) adalah resep-resep. Mereka memandu anda melalui langkah-langkah dalam mengalamatkan kunci masalah dan kasus-penggunaan. Mereka lebih ahli dari pada tutorial dan menganggap beberapa pengetahuan dari bagaimana Django bekerja.
