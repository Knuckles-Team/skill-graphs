## 本ドキュメントの構成[¶](https://docs.djangoproject.com/ja/6.0/#how-the-documentation-is-organized "Link to this heading")
Django にはドキュメントがたくさんあります。あらかじめドキュメントの構成の大まかな概要を知っておけば、特定のトピックを探すときに、どこを探せばいいかすぐに分かるようになるでしょう。
  * [チュートリアル](https://docs.djangoproject.com/ja/6.0/intro/) では、一連のステップを通して、実際に手を動かしながら Web アプリケーションを作ります。もし Django や Web アプリケーションの開発が初めてなら、ここから読み始めてください。「 [最初のステップ](https://docs.djangoproject.com/ja/6.0/#index-first-steps) 」のセクションも読んでみてください。
  * [トピックガイド](https://docs.djangoproject.com/ja/6.0/topics/) では、重要なトピックや概念について大まかに解説し、役に立つ背景情報や説明を行っています。
  * [リファレンスガイド](https://docs.djangoproject.com/ja/6.0/ref/) は、API や他の側面から見た Django の仕組みの技術的なリファレンスです。ここでは、Django の仕組みとその使い方について説明していますが、重要な概念については基本的に理解していることを前提に書かれています。
  * [How-to ガイド](https://docs.djangoproject.com/ja/6.0/howto/) はレシピ集です。よくある問題やユースケースに対して、どんなステップを踏んで解決すればいいかを説明しています。チュートリアルより発展的な内容で、Django の仕組みについてはある程度は理解していることを前提に書いてあります。
