## MariaDB notes[¶](https://docs.djangoproject.com/en/5.0/ref/databases/#mariadb-notes "Link to this heading")
Django supports MariaDB 10.4 and higher.
To use MariaDB, use the MySQL backend, which is shared between the two. See the [MySQL notes](https://docs.djangoproject.com/en/5.0/ref/databases/#mysql-notes) for more details.
