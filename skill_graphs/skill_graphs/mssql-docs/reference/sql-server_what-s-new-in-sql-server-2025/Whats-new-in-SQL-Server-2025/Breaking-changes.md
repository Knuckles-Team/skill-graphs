## Breaking changes
SQL Server 2025 (17.x) introduces breaking changes to a few SQL Server Database Engine features, such as linked servers, replication, log shipping, and PolyBase.
For more information, see [Breaking changes to Database Engine features in SQL Server 2025](https://learn.microsoft.com/en-us/sql/database-engine/breaking-changes-to-database-engine-features-in-sql-server-2025?view=sql-server-ver17).
