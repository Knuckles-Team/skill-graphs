## v0.13.0 (2018-08-25)[¶](https://docs.pydantic.dev/latest/changelog/#v0130-2018-08-25)
  * raise an exception if a field's name shadows an existing `BaseModel` attribute
  * add `UrlStr` and `urlstr` types
  * timedelta json encoding ISO8601 and total seconds, custom json encoders
  * allow `timedelta` objects as values for properties of type `timedelta` (matches `datetime` etc. behavior)
