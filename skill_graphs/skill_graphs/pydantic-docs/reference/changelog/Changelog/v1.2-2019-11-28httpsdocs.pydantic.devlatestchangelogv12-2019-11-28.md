## v1.2 (2019-11-28)[¶](https://docs.pydantic.dev/latest/changelog/#v12-2019-11-28)
  * **Possible Breaking Change:** Add support for required `Optional` with `name: Optional[AnyType] = Field(...)` and refactor `ModelField` creation to preserve `required` parameter value, [here](https://docs.pydantic.dev/usage/models/#required-optional-fields) for details
  * Add benchmarks for `cattrs`,
  * Add `exclude_none` option to `dict()` and friends,
  * Add benchmarks for `valideer`,
  * Add `parse_obj_as` and `parse_file_as` functions for ad-hoc parsing of data into arbitrary pydantic-compatible types,
  * Add `allow_reuse` argument to validators, thus allowing validator reuse,
  * Add support for mapping types for custom root models,
  * Mypy plugin support for dataclasses,
  * Add support for dataclasses default factory,
  * Add a `ByteSize` type for converting byte string (`1GB`) to plain bytes,
  * Fix mypy complaint about `@root_validator(pre=True)`,
  * Add manylinux binaries for Python 3.8 to pypi, also support manylinux2010,
  * Adds ByteSize conversion to another unit,
  * Fix `__str__` and `__repr__` inheritance for models,
  * add testimonials section to docs,
  * Add support for `typing.Literal` for Python 3.8,
