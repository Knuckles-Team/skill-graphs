## v0.19.0 (2019-02-04)[¶](https://docs.pydantic.dev/latest/changelog/#v0190-2019-02-04)
  * Support `Callable` type hint, fix
  * Fix schema for fields with `validator` decorator, fix
  * Add `multiple_of` constraint to `ConstrainedDecimal`, `ConstrainedFloat`, `ConstrainedInt` and their related types `condecimal`, `confloat`, and `conint`
  * Deprecated `ignore_extra` and `allow_extra` Config fields in favor of `extra`,
  * Add type annotations to all functions, test fully with mypy,
  * fix for 'missing' error with `validate_all` or `validate_always`,
  * Change the second/millisecond watershed for date/datetime parsing to `2e10`,
