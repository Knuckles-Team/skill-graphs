## v0.32 (2019-08-06)[¶](https://docs.pydantic.dev/latest/changelog/#v032-2019-08-06)
  * add model name to `ValidationError` error message,
  * **breaking change** : remove `__getattr__` and rename `__values__` to `__dict__` on `BaseModel`, deprecation warning on use `__values__` attr, attributes access speed increased up to 14 times,
  * support `ForwardRef` (without self-referencing annotations) in Python 3.6,
  * implement `schema_extra` in `Config` sub-class,
