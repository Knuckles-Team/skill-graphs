## v1.7.3 (2020-11-30)[¶](https://docs.pydantic.dev/latest/changelog/#v173-2020-11-30)
Thank you to pydantic's sponsors:
  * fix: set right default value for required (optional) fields,
  * fix: support `underscore_attrs_are_private` with generic models,
  * fix: update all modified field values in `root_validator` when `validate_assignment` is on,
  * Allow pickling of `pydantic.dataclasses.dataclass` dynamically created from a built-in `dataclasses.dataclass`,
  * Fix a regression where Enum fields would not propagate keyword arguments to the schema,
  * Ignore `__doc__` as private attribute when `Config.underscore_attrs_are_private` is set,
