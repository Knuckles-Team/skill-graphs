## Redefining developer experience
Vite makes web development enjoyable again
#####  Instant Server Start
On demand source file serving over native ESM, with blazing fast dependency pre-bundling.
![](https://vite.dev/assets/vite-featurepanel-1-background.BHnDg8j4.jpg)![Terminal](https://vite.dev/assets/vite-featurepanel-1-terminal.CQVUoUSU.svg)
##### Lightning Fast HMR
Instantly reflect changes as you save, no matter how big your app is.
![lightning fast hot module replacement](https://vite.dev/assets/vite-featurepanel-2-terminal.CjMck4X1.png)
##### Rich Features Out of the Box
TypeScript, JSX, CSS, Workers, Web Assembly... and more with just a plugin away.
##### Optimized Build
Advanced tree-shaking, built-in minification, fine-grained chunking control powered by Rolldown.
![](https://vite.dev/assets/vite-featurepanel-4-background.Bnx0RZk-.jpg)![optimized build](https://vite.dev/assets/vite-featurepanel-4-terminal.BRnxuy_J.svg)
## A shared foundation to build upon
#####  Flexible Plugin System
Vite plugins extends Rollup's well-designed plugin interface with a few extra Vite-specific options.
##### Fully Typed API
Designed to be built on top of.
![](https://vite.dev/assets/vite-featurepanel-4-background.Bnx0RZk-.jpg)![typed api](https://vite.dev/assets/vite-typed-api.B7gyuuaS.svg)
##### First class SSR Support
It's never been easier to setup custom SSR (Server-Side Rendering), or build your own SSR framework.
![SSR Support](https://vite.dev/assets/vite-ssr-support.C71yCKer.png)
##### Continuous ecosystem integration
Our CI continuously tests Vite changes against downstream projects, allowing us to improve Vite with stability and confidence.
![continuous ecosystem integration](https://vite.dev/assets/vite-ci.rfkpyz0p.svg)
