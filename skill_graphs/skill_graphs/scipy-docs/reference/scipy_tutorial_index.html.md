## Subpackages and User Guides[#](https://docs.scipy.org/doc/scipy/tutorial/index.html#subpackages-and-user-guides "Link to this heading")

`linalg` | [Linear Algebra (scipy.linalg)](https://docs.scipy.org/doc/scipy/tutorial/linalg.html)
`ndimage` | [Multidimensional Image Processing (scipy.ndimage)](https://docs.scipy.org/doc/scipy/tutorial/ndimage.html)
`odr` | Orthogonal distance regression
`optimize` | [Optimization (scipy.optimize)](https://docs.scipy.org/doc/scipy/tutorial/optimize.html)
