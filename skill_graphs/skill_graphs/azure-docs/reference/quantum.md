# Microsoft Quantum documentation
Learn about quantum computing with Microsoft Quantum and the Azure Quantum service. Use Python and Q#, a language for quantum programming, to create and submit quantum programs in the Azure portal, or set up your own local development environment with the Microsoft Quantum Development Kit (QDK) to write quantum programs.
