## Services
### Distribute
Quickly release Android, iOS, macOS, and UWP builds to end user devices.
  * [Introduction to App Center Distribute](https://learn.microsoft.com/en-us/appcenter/distribution/)
  * [Release a Build](https://learn.microsoft.com/en-us/appcenter/distribution/uploading)
  * [iOS Device registration](https://learn.microsoft.com/en-us/appcenter/distribution/auto-provisioning)
  * [Manage Distribution Groups](https://learn.microsoft.com/en-us/appcenter/distribution/groups)
  * [Distribute to Store](https://learn.microsoft.com/en-us/appcenter/distribution/stores/)


### Analytics
Get information about the number of users, the session duration, top devices and more.
  * [Introduction to Analytics](https://learn.microsoft.com/en-us/appcenter/analytics/)
  * [Analytics Overview Page](https://learn.microsoft.com/en-us/appcenter/analytics/overview)
  * [Events Metrics](https://learn.microsoft.com/en-us/appcenter/analytics/event-metrics)
  * [FAQ](https://learn.microsoft.com/en-us/appcenter/analytics/faq)


### Diagnostics
Collect crash data from all devices.
  * [Introduction to Diagnostics](https://learn.microsoft.com/en-us/appcenter/diagnostics/)
  * [Enable Diagnostics in an App](https://learn.microsoft.com/en-us/appcenter/diagnostics/enabling-diagnostics)
  * [Diagnostics Features](https://learn.microsoft.com/en-us/appcenter/diagnostics/features)
  * [Troubleshooting](https://learn.microsoft.com/en-us/appcenter/diagnostics/troubleshooting)
