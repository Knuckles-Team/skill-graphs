# Learn about Azure AI Video Indexer
Azure AI Video Indexer is an AI solution that enables you to extract insights from your videos using video and audio models. It's available as an Azure Arc extension for edge deployment or as a cloud-based application, part of Azure AI services built on services such as Face, Translator, Azure AI Vision, and Speech.
