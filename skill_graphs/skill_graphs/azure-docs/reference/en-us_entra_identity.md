# Microsoft Entra ID documentation
Use Microsoft Entra ID to manage user identities and control access to your apps, data, and resources.
[](https://learn.microsoft.com/en-us/entra/identity/#getting-started)
## Getting started
![](https://learn.microsoft.com/en-us/entra/identity/media/index/entra-id.svg) [What is Microsoft Entra ID?](https://learn.microsoft.com/en-us/entra/fundamentals/what-is-entra)
Manage user identities and control access to your apps, data, and resources.
![](https://learn.microsoft.com/en-us/entra/identity/media/index/authentication.svg) [Authentication](https://learn.microsoft.com/en-us/entra/identity/authentication/overview-authentication)
Configure sign-in methods and security features like self-service password reset and multifactor authentication.
![](https://learn.microsoft.com/en-us/entra/identity/media/index/enterprise-applications.svg) [Application management](https://learn.microsoft.com/en-us/entra/identity/enterprise-apps/what-is-application-management)
Develop, add, or connect an app to Microsoft Entra ID and manage access.
![](https://learn.microsoft.com/en-us/entra/identity/media/index/role-based-access-control.svg) [Role-based access control](https://learn.microsoft.com/en-us/entra/identity/role-based-access-control/custom-overview)
Manage administrator permissions and apply the principle of least privilege using Microsoft Entra role-based access control.
![](https://learn.microsoft.com/en-us/entra/identity/media/index/users.svg) [User management](https://learn.microsoft.com/en-us/entra/identity/users/directory-overview-user-model)
Manage users and groups, assign licenses, and manage domain names.
![](https://learn.microsoft.com/en-us/entra/identity/media/index/conditional-access.svg) [Conditional Access](https://learn.microsoft.com/en-us/entra/identity/conditional-access/overview)
Control access to resources by enforcing policies based on user, location, device, and more.
![](https://learn.microsoft.com/en-us/entra/identity/media/index/devices.svg) [Device identity](https://learn.microsoft.com/en-us/entra/identity/devices/overview)
Register and join devices to Microsoft Entra ID for device management and Condition Access.
![](https://learn.microsoft.com/en-us/entra/identity/media/index/hybrid-identity.svg) [Hybrid identity](https://learn.microsoft.com/en-us/entra/identity/hybrid/whatis-hybrid-identity)
Create a user identity that can access both on-premises and cloud resources.
![](https://learn.microsoft.com/en-us/entra/identity/media/index/application-provisioning.svg) [Application provisioning](https://learn.microsoft.com/en-us/entra/identity/app-provisioning/user-provisioning)
Create and manage user identities in applications by provisioning from Microsoft Entra ID.
![](https://learn.microsoft.com/en-us/entra/identity/media/index/enterprise-applications.svg) [Application proxy](https://learn.microsoft.com/en-us/entra/identity/app-proxy/overview-what-is-app-proxy)
Provide secure remote access and cloud scale security to your private applications.
![](https://learn.microsoft.com/en-us/entra/identity/media/index/managed-identities.svg) [Managed identities for Azure resources](https://learn.microsoft.com/en-us/entra/identity/managed-identities-azure-resources/overview)
Create an application identity that can connect to resources using Microsoft Entra authentication.
![](https://learn.microsoft.com/en-us/entra/identity/media/index/enterprise-applications.svg) [Application integration](https://learn.microsoft.com/en-us/entra/identity/saas-apps/tutorial-list)
Integrate cloud-enabled, software as a service (SaaS) apps with Microsoft Entra ID.
![](https://learn.microsoft.com/en-us/entra/identity/media/index/monitoring-health.svg) [Monitoring and health](https://learn.microsoft.com/en-us/entra/identity/monitoring-health/overview-monitoring-health)
Use logs and reports to determine usage of your apps and services, detect risks, and troubleshoot issues.
![](https://learn.microsoft.com/en-us/entra/identity/media/index/multitenant-organizations.svg) [Multitenant organizations](https://learn.microsoft.com/en-us/entra/identity/multi-tenant-organizations/overview)
Collaborate across tenants within your organization.
![](https://learn.microsoft.com/en-us/entra/identity/media/index/domain-services.svg) [Domain services](https://learn.microsoft.com/en-us/entra/identity/domain-services/overview)
Move legacy apps to managed domains in the cloud while preserving user accounts, groups, and access.
![](https://learn.microsoft.com/en-us/entra/identity/media/index/azure-ad-b2c.svg) [Azure Active Directory B2C](https://learn.microsoft.com/en-us/azure/active-directory-b2c/)
Business to consumers identity and access management for your app.
[](https://learn.microsoft.com/en-us/entra/identity/#resources)
