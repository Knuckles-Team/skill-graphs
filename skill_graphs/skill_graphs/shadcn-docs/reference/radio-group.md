# Radio Group
[Previous](https://ui.shadcn.com/docs/components/radix/progress)[Next](https://ui.shadcn.com/docs/components/radix/resizable)
A set of checkable buttons—known as radio buttons—where no more than one of the buttons can be checked at a time.
