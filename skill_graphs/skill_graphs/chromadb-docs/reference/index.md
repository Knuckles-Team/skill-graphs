# Introduction
Copy page
Chroma is an open-source search engine for AI. It comes with everything you need to get started built-in.
Copy page
Chroma gives you everything you need for retrieval: store embeddings with metadata, search with dense and sparse vectors, filter by metadata, and retrieve across text, images, and more.
##
[​](https://docs.trychroma.com/docs/overview/introduction#what-chroma-offers)
What Chroma Offers
## [Document Storage Store documents and metadata. ](https://docs.trychroma.com/docs/collections/add-data)## [Embeddings Use any embedding model. OpenAI, Cohere, Hugging Face, sentence-transformers, and more. ](https://docs.trychroma.com/docs/embeddings/embedding-functions)## [Vector Search Dense, sparse, and hybrid search. Query by similarity and combine multiple search strategies. ](https://docs.trychroma.com/docs/querying-collections/query-and-get)## [Full-Text & Regex Search Keyword and regex search over your data without embeddings. ](https://docs.trychroma.com/docs/querying-collections/full-text-search)## [Metadata Filtering Filter results at query time by metadata conditions. ](https://docs.trychroma.com/docs/querying-collections/metadata-filtering)## [Multi-Modal Retrieval Index and search images, audio, and other modalities alongside text. ](https://docs.trychroma.com/docs/embeddings/multimodal)
