The Chroma team regularly releases sample AI applications powered by Chroma, which you can use to learn about retrieval, building with AI, and as a jumping-off board for your own projects. The CLI makes it easy to install and set up the Chroma sample apps on your local machine with the `chroma install` command. To install a sample app simply run
