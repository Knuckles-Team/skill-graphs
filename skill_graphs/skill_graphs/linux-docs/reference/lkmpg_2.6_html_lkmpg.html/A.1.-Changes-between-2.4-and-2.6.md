##  A.1.1. Changes between 2.4 and 2.6
I don't know the entire kernel well enough to document all of the changes. Some hints for porting can be found by comparing this version of the LKMPG with it's counterpart for kernel 2.4. Apart from that, anybody who needs to port drivers from 2.4 to 2.6 kernels might want to visit **xxdiff** or **meld** can be a great help then. Also check if your driver is covered by docs in ` linux/Documentation/ `. Before starting with porting and in case you're stuck it's a good idea to find an appropriate mailinglist and ask people there for pointers.
