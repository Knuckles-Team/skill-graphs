## Step 2: Install Cacti Tool in Linux
To install **Cacti** , make sure that the [EPEL repository](https://www.tecmint.com/install-epel-repo-rhel-rocky-almalinux/ "Install EPEL Repository") is enabled on your system.
```
sudo yum install epel-release

```

Once the EPEL repository is enabled, you can install Cacti by running the following command.
```
sudo yum install cacti

```
