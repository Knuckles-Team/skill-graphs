###  Linux Documentation Project Guides

The Linux Documentation Project (LDP) is working on developing good, reliable documentation for the Linux operating system. The overall goal of the LDP is to collaborate in taking care of all of the issues of Linux documentation, ranging from online documentation (man pages, HTML, and so on) to printed manuals covering topics such as installing, using, and running Linux. Here is the Linux Documentation Project [ Manifesto](https://tldp.org/LDP-Manifesto.html) and [Copyright License](https://tldp.org/LDP-COPYRIGHT.html) for LDP works.  **Translations of LDP works** (languages other than English) can be found on the "[Non-English Linux Info](https://tldp.org/links/nenglish.html)" links page.  A subset of [**Debian manuals**](https://tldp.org/guides.html#debian) are now available.  **Additional documentation** (HOWTOs, FAQs, etc) can be found [here](https://tldp.org/docs.html).  **Older and unmaintained (but still pertinent) guides** can be found [here](https://tldp.org/guides.html#older).  **Search the LDP documents** from our [search page](https://tldp.org/search.html).
