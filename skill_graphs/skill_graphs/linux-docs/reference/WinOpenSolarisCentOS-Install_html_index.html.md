> * * *
> _This document exemplifies how three different operating systems (OSes) could be installed and configured on a single hard disk. The reader may choose from: Microsoft Windows NT/2K/XP + OpenSolaris 2008.11 + CentOS 5.2. Only x86 & x86-compatible architecture multi-booting is discussed and GNU GRUB is the boot loader of choice. _ _Drop an email at_ _in case of queries/questions/comments. Thanks!_**
