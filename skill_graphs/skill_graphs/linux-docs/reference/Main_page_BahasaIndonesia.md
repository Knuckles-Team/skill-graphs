## [Kabar Terbaru Wiki](https://wiki.archlinux.org/title/Wiki_News "Wiki News")
Cek halaman _[Wiki News](https://wiki.archlinux.org/title/Wiki_News "Wiki News")_ untuk mengetahui pembaruan wiki.
Jika berminat dan mampu berkontribusi ke wiki, silakan lihat [daftar ini](https://wiki.archlinux.org/title/Getting_involved_\(Indonesia\)#Berkontribusi_untuk_Arch_Wiki "Getting involved \(Indonesia\)") untuk lebih jelasnya. Khususnya, mereka yang butuh bantuan mengatur ulang kategori non-bahasa Inggris. Silakan koordinasikan aktivitas anda dengan [admin dan _sysops_](https://wiki.archlinux.org/title/ContactList "ContactList").

Anda harus membuat sebuah akun untuk mengedit wiki ini. Setelah memilikinya, anda dapat menelusuri _[Sandbox](https://wiki.archlinux.org/title/Sandbox "Sandbox")_ atau halaman pengguna anda untuk membiasakan diri dengan penulisan kode (_markup_) ArchWiki. Syntax yang bisa digunakan ada di [Kategori Bantuan Wiki](https://wiki.archlinux.org/title/Category:Help "Category:Help") bersama dengan sumber bermanfaat lain.
