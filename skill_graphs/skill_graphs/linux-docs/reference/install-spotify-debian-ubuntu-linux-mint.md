**Spotify** for Linux is available as a Debian `.deb` package which you can install on [Debian and Ubuntu Linux](https://www.tecmint.com/debian-based-linux-distributions/ "Best Debian-based Linux Distributions") and its derivatives.
First, add the **Spotify Debian** repository key by running the following [curl command](https://www.tecmint.com/linux-curl-command-examples/ "How to Use Curl Command in Linux").
```
$ curl -sS https://download.spotify.com/debian/pubkey_7A3A762FAFD4A51F.gpg | sudo gpg --dearmor --yes -o /etc/apt/trusted.gpg.d/spotify.gpg

```

Next, add the repository configuration to your systems package manager configuration:
```
$ echo "deb http://repository.spotify.com stable non-free" | sudo tee /etc/apt/sources.list.d/spotify.list

```

Now update the local package cache to fetch the list of packages from newly added repositories.
```
$ sudo apt-get update

```

Then install the **Spotify** client with the following command.
```
$ sudo apt-get install spotify-client

```

**Spotify** is also available as a [snap](https://www.tecmint.com/install-snap-in-linux/ "Install Snaps in Linux") (applications packaged with all their dependencies to run on all [popular Linux distributions](https://www.tecmint.com/top-most-popular-linux-distributions/ "Most Popular Linux Distributions") from a single build), which you can install as follows.
To run **snap** applications, you need:
  * **snapd** (**snap daemon**) – the background service that manages and maintains the snaps on a Linux system, and.
  * **snap** – a command-line tool used to interact with snaps, which allows you to install, configure, refresh, and uninstall/remove snaps on a Linux system.


Now run the following commands to install the **snapd** package and **Spotify** from the
```
$ sudo apt update
$ sudo apt install snapd
$ sudo snap install spotify

```

From the system menu, search for the **Software Manager** application and open it. Once it opens, search for the **Spotify** application and click on it.
![Search Spotify in Software Manager](https://www.tecmint.com/wp-content/uploads/2023/03/Search-Spotify-in-Software-Manager.png)Search Spotify in Software Manager
Next, click on the **Spotify** app as highlighted in the above screenshot. Next, click on the **Install** button as highlighted in the following screenshot, to install it.
![Install Spotify via Software Manager](https://www.tecmint.com/wp-content/uploads/2023/03/Install-Spotify-via-Software-Manager.png)Install Spotify via Software Manager
Next, enter your account password and click on the **Authenticate** button to start the installation process.
Once **Spotify** is installed on your system, search for it in the system menu. Then open it by clicking on it. After it opens, log in if you already have an account or sign up (create an account).
![Spotify Login](https://www.tecmint.com/wp-content/uploads/2023/03/Spotify-Login.png)Spotify Login
There are several login options including username/email and password, or Facebook, Google, or Apple.
![Spotify Login Options](https://www.tecmint.com/wp-content/uploads/2023/03/Spotify-Login-Options.png)Spotify Login Options
After a successful login, you can now play and listen to your favorite songs and albums for free on Linux with Spotify. To download and listen to music offline without ads, wherever you are, you need to subscribe to **Spotify Premium**.
![Spotify Home](https://www.tecmint.com/wp-content/uploads/2023/03/Spotify-Dashboard.png)Spotify Home
