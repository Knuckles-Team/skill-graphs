#  E.2. OpenOffice.org to DocBook
As of
Although OOo uses the full DocBook document type declaration, it does not actually export the full list of DocBook elements. It uses a "simplified" DocBook tag set which is geared to on-the-fly rendering. (Although it is not the official Simplified DocBook which is described in [Section B.5](https://tldp.org/LDP/LDP-Author-Guide/html/LDP-Author-Guide.html#dtd).)
