##  3.1.4. Cross validation and model selection[#](https://scikit-learn.org/stable/modules/cross_validation.html#cross-validation-and-model-selection "Link to this heading")
Cross validation iterators can also be used to directly perform model selection using Grid Search for the optimal hyperparameters of the model. This is the topic of the next section: [Tuning the hyper-parameters of an estimator](https://scikit-learn.org/stable/modules/grid_search.html#grid-search).
