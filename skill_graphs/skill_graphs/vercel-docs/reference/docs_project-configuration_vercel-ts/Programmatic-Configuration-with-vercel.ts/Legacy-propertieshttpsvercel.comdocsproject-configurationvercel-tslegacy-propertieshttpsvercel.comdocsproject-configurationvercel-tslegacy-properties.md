##  [Legacy properties](https://vercel.com/docs/project-configuration/vercel-ts#legacy-properties)[](https://vercel.com/docs/project-configuration/vercel-ts#legacy-properties)
Legacy properties like `routes` and `builds` are still supported in `vercel.ts` for backwards compatibility, but are deprecated. We recommend using the helper-based options above (`rewrites`, `redirects`, `headers`) for type safety and better developer experience.
For details on legacy properties, see the [legacy section of the static configuration reference](https://vercel.com/docs/project-configuration/vercel-json#legacy).
