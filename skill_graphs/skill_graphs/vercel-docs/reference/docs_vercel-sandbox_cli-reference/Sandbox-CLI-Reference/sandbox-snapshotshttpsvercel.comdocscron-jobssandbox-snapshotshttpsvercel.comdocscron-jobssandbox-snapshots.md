##  [`sandbox snapshots`](https://vercel.com/docs/cron-jobs#sandbox-snapshots)[](https://vercel.com/docs/cron-jobs#sandbox-snapshots)
Manage sandbox snapshots.
terminal
```
sandbox snapshots <subcommand> [OPTIONS]
```

###  [Sandbox snapshots subcommands](https://vercel.com/docs/cron-jobs#sandbox-snapshots-subcommands)[](https://vercel.com/docs/cron-jobs#sandbox-snapshots-subcommands)
  * `list`: List snapshots for the specified account and project. [alias: `ls`]
  * `get`: Get details of a snapshot.
  * `delete`: Delete one or more snapshots. [aliases: `rm`, `remove`]
