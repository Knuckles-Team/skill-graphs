##  [Installation](https://vercel.com/docs/cron-jobs#installation)[](https://vercel.com/docs/cron-jobs#installation)
Install the Sandbox CLI globally to use all commands:
Terminal
![](https://7nyt0uhk7sse4zvn.public.blob.vercel-storage.com/docs-assets/static/topics/icons/pnpm.svg)pnpm bun yarn npm
```
pnpm i -g sandbox
```

```
yarn global add sandbox
```

```
npm i -g sandbox
```

```
bun add -g sandbox
```

You can invoke the CLI using the `sandbox` or `sbx` commands in your terminal.
