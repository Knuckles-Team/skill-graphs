# Web Search
Last updated February 7, 2026
AI Gateway provides built-in web search capabilities that allow AI models to access current information from the web. This is useful when you need up-to-date information that may not be in the model's training data.
AI Gateway supports two types of web search:
  * Search for all providers: Use [Perplexity Search](https://vercel.com/docs/ai-gateway/agent-quickstart#using-perplexity-search) or [Parallel Search](https://vercel.com/docs/ai-gateway/agent-quickstart#using-parallel-search) with any model regardless of provider. This gives you consistent web search behavior across different models.
  * Provider-specific search: Use native web search tools from [Anthropic](https://vercel.com/docs/ai-gateway/agent-quickstart#anthropic-web-search), [OpenAI](https://vercel.com/docs/ai-gateway/agent-quickstart#openai-web-search), or [Google](https://vercel.com/docs/ai-gateway/agent-quickstart#google-web-search). These tools are optimized for their respective providers and may offer [additional features](https://vercel.com/docs/ai-gateway/agent-quickstart#provider-specific-search).
