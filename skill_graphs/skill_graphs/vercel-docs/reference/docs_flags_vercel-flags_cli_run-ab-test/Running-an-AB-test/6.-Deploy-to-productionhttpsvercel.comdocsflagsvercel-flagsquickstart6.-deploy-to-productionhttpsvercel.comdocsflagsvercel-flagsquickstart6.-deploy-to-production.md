##  [6. Deploy to production](https://vercel.com/docs/flags/vercel-flags/quickstart#6.-deploy-to-production)[](https://vercel.com/docs/flags/vercel-flags/quickstart#6.-deploy-to-production)
terminal
```
vercel deploy --prod
```

Enable the flag in the Production environment in the dashboard to start serving the new layout to users. Monitor results in Web Analytics by comparing metrics per variant.
