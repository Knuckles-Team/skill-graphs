##  [5. Deploy to preview and test both variants](https://vercel.com/docs/flags/vercel-flags/quickstart#5.-deploy-to-preview-and-test-both-variants)[](https://vercel.com/docs/flags/vercel-flags/quickstart#5.-deploy-to-preview-and-test-both-variants)

```
vercel deploy
```

Visit the preview URL to confirm both layouts render correctly. If you've set up the [Flags Explorer](https://vercel.com/docs/flags/flags-explorer), you can toggle the flag in the toolbar.
