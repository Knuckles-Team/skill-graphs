# Build with AI agents on Vercel
Last updated February 3, 2026
Integrating AI agents in your application often means working with separate dashboards, billing systems, and authentication flows for each agent you want to use. This can be time-consuming and frustrating.
With [AI agents](https://vercel.com/docs/agent-resources/integrations-for-agents#ai-agents) and [AI agent services](https://vercel.com/docs/agent-resources/integrations-for-agents#ai-agent-services) on the Vercel Marketplace, you can add AI-powered workflows to your projects through [native integrations](https://vercel.com/docs/integrations#native-integrations) and get a unified dashboard with billing, observability, and installation flows.
You have access to two types of AI building blocks:
  * [Agents](https://vercel.com/docs/agent-resources/integrations-for-agents#ai-agents): Pre-built systems that handle specialized workflows on your behalf
  * [Services](https://vercel.com/docs/agent-resources/integrations-for-agents#ai-agent-services): Infrastructure you use to build and run your own agents
