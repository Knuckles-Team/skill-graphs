##  [Upload charges](https://vercel.com/docs/vercel-blob/public-storage#upload-charges)[](https://vercel.com/docs/vercel-blob/public-storage#upload-charges)
Upload charges depend on your implementation method:
  * [Client Uploads](https://vercel.com/docs/vercel-blob/client-upload): No data transfer charges for uploads
  * [Server Uploads](https://vercel.com/docs/vercel-blob/server-upload): [Fast Data Transfer](https://vercel.com/docs/manage-cdn-usage#fast-data-transfer) charges apply when your Vercel application receives the file


See [pricing documentation](https://vercel.com/docs/vercel-blob/usage-and-pricing) for full details.
