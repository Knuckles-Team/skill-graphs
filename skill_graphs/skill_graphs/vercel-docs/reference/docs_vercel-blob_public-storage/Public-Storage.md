# Public Storage
Last updated March 8, 2026
Vercel Blob is available on [all plans](https://vercel.com/docs/plans)
Those with the [owner, member, developer](https://vercel.com/docs/rbac/access-roles#owner,%20member,%20developer-role) role can access this feature
Public Blob stores make files accessible to anyone with the URL. Use public storage for images, videos, large media, and public assets where authentication isn't needed.
See [differences with private storage](https://vercel.com/docs/vercel-blob#private-and-public-storage).
