##  [Prerequisites](https://vercel.com/docs/tracing/session-tracing#prerequisites)[](https://vercel.com/docs/tracing/session-tracing#prerequisites)
  * A Vercel account. If you don't have one, you can [sign up for free](https://vercel.com/signup).
  * A Vercel project that is deployed to preview or production. You cannot create and run a session trace for a local deployment.
  * [The toolbar enabled](https://vercel.com/docs/vercel-toolbar/in-production-and-localhost) in your preview or production environment.
