# Using the Flags SDK with Vercel Flags
Last updated March 8, 2026
The [Flags SDK](https://vercel.com/docs/flags/flags-sdk-reference) is the recommended way to use Vercel Flags in Next.js and SvelteKit applications. It provides a framework-native experience with full TypeScript support and automatic integration with Flags Explorer.
The [Getting Started guide](https://vercel.com/docs/flags/vercel-flags/quickstart) covers installing packages, pulling environment variables, defining a flag, and evaluating it in a component. This page builds on that and goes deeper into the adapter, Flags Explorer, targeting, and SvelteKit.
