# Managing Code Reviews
Last updated October 23, 2025
Once you've [set up Code Review](https://vercel.com/docs/agent/pr-review#how-to-set-up-code-review), you can customize settings and monitor performance from the [Agent section in the sidebar](https://vercel.com/d?to=%2F%5Bteam%5D%2F%7E%2Fvercel-agent&title=Open+Vercel+Agent) in your dashboard. This is your central hub for managing which repositories get reviewed, tracking costs, and analyzing how reviews are performing.
