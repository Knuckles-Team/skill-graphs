##  [Accessing the toolbar using the toolbar menu](https://vercel.com/docs/comments#accessing-the-toolbar-using-the-toolbar-menu)[](https://vercel.com/docs/comments#accessing-the-toolbar-using-the-toolbar-menu)
Provided [the Vercel toolbar is enabled](https://vercel.com/docs/vercel-toolbar/managing-toolbar#enable-or-disable-the-toolbar-project-wide) for your project, you can enable the toolbar on production environments from the toolbar menu:
  1. Open a preview deployment of your project.
  2. Select the menu icon in the toolbar.
  3. Scroll down to Enable Vercel Toolbar in Production and select it.
  4. Choose the domain you want to enable the toolbar on.
