##  [schema autocomplete](https://vercel.com/docs/project-configuration/vercel-json#schema-autocomplete)[](https://vercel.com/docs/project-configuration/vercel-json#schema-autocomplete)
To add autocompletion, type checking, and schema validation to your `vercel.json` file, add the following to the top of your file:
```
{
  "$schema": "https://openapi.vercel.sh/vercel.json"
}
```
