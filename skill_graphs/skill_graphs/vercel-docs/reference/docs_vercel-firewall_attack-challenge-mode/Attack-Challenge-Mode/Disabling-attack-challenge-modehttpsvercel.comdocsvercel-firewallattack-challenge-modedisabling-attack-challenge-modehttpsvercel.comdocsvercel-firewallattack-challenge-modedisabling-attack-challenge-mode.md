##  [Disabling attack challenge mode](https://vercel.com/docs/vercel-firewall/attack-challenge-mode#disabling-attack-challenge-mode)[](https://vercel.com/docs/vercel-firewall/attack-challenge-mode#disabling-attack-challenge-mode)
When you no longer need the additional protection:
  1. Select your project from the [Dashboard](https://vercel.com/dashboard)
  2. Open [Firewall](https://vercel.com/d?to=%2F%5Bteam%5D%2F%5Bproject%5D%2Ffirewall&title=Go+to+Firewall) in the sidebar.
  3. Click Bot Management.
  4. Under Attack Challenge Mode, select Disable.
