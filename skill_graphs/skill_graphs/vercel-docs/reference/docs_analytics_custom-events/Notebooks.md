# Notebooks
Last updated October 7, 2025
Notebooks are available on [Enterprise](https://vercel.com/docs/plans/enterprise) and [Pro](https://vercel.com/docs/plans/pro) plans
Notebooks allow you to collect and manage multiple queries related to your application's metrics and performance data.
Within a single notebook, you can store multiple queries that examine different aspects of your system - each with its own specific filters, time ranges, and data aggregations. You can build comprehensive dashboards or analysis workflows by grouping related queries together.
You need to enable [Observability Plus](https://vercel.com/docs/observability/observability-plus) to use Notebooks since you need run queries.
