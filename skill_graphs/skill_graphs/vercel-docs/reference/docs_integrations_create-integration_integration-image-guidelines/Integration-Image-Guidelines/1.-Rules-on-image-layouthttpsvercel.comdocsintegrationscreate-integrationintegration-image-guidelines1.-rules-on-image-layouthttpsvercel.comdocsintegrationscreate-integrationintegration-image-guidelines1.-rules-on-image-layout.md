##  [1. Rules on image layout](https://vercel.com/docs/integrations/create-integration/integration-image-guidelines#1.-rules-on-image-layout)[](https://vercel.com/docs/integrations/create-integration/integration-image-guidelines#1.-rules-on-image-layout)
a. Images must use a 16:9 layout (1920 × 1080 minimum).
b. Layouts must have symmetrical margins and a reasonable safe area.
c. All images must have both a central visual asset and a description.
