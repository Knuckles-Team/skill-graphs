##  [Troubleshooting](https://vercel.com/docs/deploy-hooks#troubleshooting)[](https://vercel.com/docs/deploy-hooks#troubleshooting)
If your deploy hook fails to create a deployment, check the status check on the commit associated with the deploy hook to identify any failures. See [Troubleshooting project collaboration](https://vercel.com/docs/deployments/troubleshoot-project-collaboration) for more information.
