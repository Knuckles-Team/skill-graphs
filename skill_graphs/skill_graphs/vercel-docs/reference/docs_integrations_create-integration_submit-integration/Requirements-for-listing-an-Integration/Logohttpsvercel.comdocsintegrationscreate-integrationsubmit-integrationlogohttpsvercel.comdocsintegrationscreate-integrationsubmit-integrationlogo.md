##  [Logo](https://vercel.com/docs/integrations/create-integration/submit-integration#logo)[](https://vercel.com/docs/integrations/create-integration/submit-integration#logo)
  * Required: Yes


The image displayed in a circle, that appears throughout the dashboard and marketing pages. Like all assets, it will appear in both light and dark mode.
![](https://vercel.com/vc-ap-vercel-docs/_next/image?url=https%3A%2F%2F7nyt0uhk7sse4zvn.public.blob.vercel-storage.com%2Fdocs-assets%2Fstatic%2Fdocs%2Fintegrations%2Fcreating%2Ficon.png&w=750&q=75)
You must make sure that the images adhere to the following dimensions and aspect ratios:
Spec Name | Ratio | Size | Notes
---|---|---|---
Icon | 1:1 | 20-80px | High resolution bitmap image, non-transparent PNG, minimum 256px
