# Observability Plus
Last updated January 23, 2026
Observability Plus is available on [Enterprise](https://vercel.com/docs/plans/enterprise) and [Pro](https://vercel.com/docs/plans/pro) plans
Those with the [owner](https://vercel.com/docs/rbac/access-roles#owner-role) role can access this feature
Observability Plus is an optional upgrade that enables Pro and Enterprise teams to explore data at a more granular level, helping you to pinpoint exactly when and why issues occurred.
To learn more about Observability Plus, see [Limitations](https://vercel.com/docs/observability/observability-plus#limitations) or [pricing](https://vercel.com/docs/observability/observability-plus#pricing).
