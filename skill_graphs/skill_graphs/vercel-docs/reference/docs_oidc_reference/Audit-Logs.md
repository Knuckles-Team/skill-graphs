# Audit Logs
Last updated December 19, 2025
Audit Logs are available on [Enterprise plans](https://vercel.com/docs/plans/enterprise)
Those with the [owner](https://vercel.com/docs/rbac/access-roles#owner-role) role can access this feature
Audit logs help you track and analyze your [team members'](https://vercel.com/docs/rbac/managing-team-members) activity. They can be accessed by team members with the [owner](https://vercel.com/docs/rbac/access-roles#owner-role) role, and are available to customers on [enterprise](https://vercel.com/docs/plans/enterprise) plans.
