##  [Query parameters](https://vercel.com/docs#query-parameters)[](https://vercel.com/docs#query-parameters)
fromstringRequired
Inclusive start of the date range as an ISO 8601 date-time string in UTC.
tostringRequired
Exclusive end of the date range as an ISO 8601 date-time string in UTC.
teamIdstringOptional
The Team identifier to perform the request on behalf of.
slugstringOptional
The Team slug to perform the request on behalf of.
