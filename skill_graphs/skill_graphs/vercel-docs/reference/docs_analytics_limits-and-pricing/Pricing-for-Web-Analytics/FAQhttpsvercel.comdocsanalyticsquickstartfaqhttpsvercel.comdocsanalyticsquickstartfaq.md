##  [FAQ](https://vercel.com/docs/analytics/quickstart#faq)[](https://vercel.com/docs/analytics/quickstart#faq)
###  [What is an event in Vercel Web Analytics?](https://vercel.com/docs/analytics/quickstart#what-is-an-event-in-vercel-web-analytics)[](https://vercel.com/docs/analytics/quickstart#what-is-an-event-in-vercel-web-analytics)
An event in Vercel Web Analytics is either an automatically tracked page view or a [custom event](https://vercel.com/docs/analytics/custom-events). A page view is a default event that is automatically tracked by our script when a user visits a page on your website. A custom event is any other action that you want to track on your website, such as a button click or form submission.
###  [What happens when you reach the maximum number of events?](https://vercel.com/docs/analytics/quickstart#what-happens-when-you-reach-the-maximum-number-of-events)[](https://vercel.com/docs/analytics/quickstart#what-happens-when-you-reach-the-maximum-number-of-events)
  * Hobby teams won't be billed beyond their allocation. Instead, collection will be paused after the 3 days grace period.
  * Pro and Enterprise teams will be billed per collected event.


###  [Is usage shared across projects?](https://vercel.com/docs/analytics/quickstart#is-usage-shared-across-projects)[](https://vercel.com/docs/analytics/quickstart#is-usage-shared-across-projects)
Yes, events are shared across all projects under the same Vercel account in Web Analytics. This means that the events collected by each project count towards the total event limit for your account. Keep in mind that if you have high-traffic websites or multiple projects with heavy event usage, you may need to upgrade to a higher-tier plan to accommodate your needs.
###  [What is the reporting window?](https://vercel.com/docs/analytics/quickstart#what-is-the-reporting-window)[](https://vercel.com/docs/analytics/quickstart#what-is-the-reporting-window)
The reporting window in Vercel Web Analytics is the length of time that your analytics data is guaranteed to be stored and viewable for analysis. While only the reporting window is guaranteed to be stored, Vercel may store your data for longer periods to give you the option to upgrade to a bigger plan without losing any data.
