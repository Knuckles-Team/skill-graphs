##  [2. Deploy to production](https://vercel.com/docs/rolling-releases/rolling-release-deployment#2.-deploy-to-production)[](https://vercel.com/docs/rolling-releases/rolling-release-deployment#2.-deploy-to-production)
Create a new production deployment. With rolling releases configured, the deployment won't immediately receive all traffic:
terminal
```
vercel deploy --prod
```

Save the deployment URL from the output for use in the following steps.
