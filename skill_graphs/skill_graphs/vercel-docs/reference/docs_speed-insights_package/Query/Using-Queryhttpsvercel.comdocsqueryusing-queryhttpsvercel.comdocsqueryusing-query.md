##  [Using Query](https://vercel.com/docs/query#using-query)[](https://vercel.com/docs/query#using-query)
  * When building queries, you can select the most appropriate view, and visualize results with:
    * a line or a volume chart
    * a table, if your query has a group by clause
    * a big number (with a time series), if your query has no group by clause
  * You can [save your queries](https://vercel.com/docs/query#save-and-share-query) in [notebooks](https://vercel.com/docs/notebooks) either for personal use or to share with your team.
  * In the dashboard, you can [create a new query](https://vercel.com/docs/query#create-a-new-query) using the query [form fields](https://vercel.com/docs/query/reference#group-by-and-where-fields) or the AI assistant at top of the new query form.
  * You can export query results as CSV or JSON by clicking the download icon.
