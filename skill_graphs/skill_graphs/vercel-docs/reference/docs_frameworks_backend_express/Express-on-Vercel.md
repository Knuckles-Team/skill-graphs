# Express on Vercel
Last updated November 14, 2025
Express is a fast, unopinionated, minimalist web framework for Node.js. You can deploy an Express app to Vercel with zero configuration.
Express applications on Vercel benefit from:
  * [Fluid compute](https://vercel.com/docs/fluid-compute): Active CPU billing, automatic cold start prevention, optimized concurrency, background processing, and more
  * [Preview deployments](https://vercel.com/docs/deployments/environments#preview-environment-pre-production): Test your changes on a copy of your production infrastructure
  * [Instant Rollback](https://vercel.com/docs/instant-rollback): Recover from unintended changes or bugs in milliseconds
  * [Vercel Firewall](https://vercel.com/docs/vercel-firewall): Protect your applications from a wide range of threats with a multi-layered security system
  * [Secure Compute](https://vercel.com/docs/secure-compute): Create private links between your Vercel-hosted backend and other clouds
