# Gatsby on Vercel
Last updated December 18, 2025
Gatsby is an open-source static-site generator. It enables developers to build fast and secure websites that integrate different content, APIs, and services.
Gatsby also has a large ecosystem of plugins and tools that improve the development experience. Vercel supports many Gatsby features, including [Server-Side Rendering](https://vercel.com/docs/getting-started-with-vercel#server-side-rendering), [Deferred Static Generation](https://vercel.com/docs/getting-started-with-vercel#deferred-static-generation), [API Routes](https://vercel.com/docs/getting-started-with-vercel#api-routes), and more.
