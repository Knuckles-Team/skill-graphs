# WAF Custom Rules
Last updated January 22, 2026
You can [configure](https://vercel.com/docs/vercel-firewall/vercel-waf/custom-rules#custom-rule-configuration) specific rules to log, deny, challenge, bypass, or [rate limit](https://vercel.com/docs/security/vercel-waf/rate-limiting) traffic to your site. When you apply the configuration, it takes effect immediately and does not require re-deployment.
[Get started](https://vercel.com/docs/vercel-firewall/vercel-waf/custom-rules#get-started) by reviewing the [Best practices for applying rules](https://vercel.com/docs/vercel-firewall/vercel-waf/custom-rules#best-practices-for-applying-rules) section.
WAF Custom Rules are available on [all plans](https://vercel.com/docs/plans)
Those with the [member](https://vercel.com/docs/rbac/access-roles#member-role), [viewer](https://vercel.com/docs/rbac/access-roles#viewer-role), [developer](https://vercel.com/docs/rbac/access-roles#developer-role) and [administrator](https://vercel.com/docs/rbac/access-roles#project-administrators) roles can access this feature
