##  [Deploy Hooks](https://vercel.com/docs/project-configuration/git-settings#deploy-hooks)[](https://vercel.com/docs/project-configuration/git-settings#deploy-hooks)
Vercel supports deploy hooks, which are unique URLs that accept HTTP POST requests and trigger deployments. Check out [our Deploy Hooks documentation](https://vercel.com/docs/deploy-hooks) to learn more.
