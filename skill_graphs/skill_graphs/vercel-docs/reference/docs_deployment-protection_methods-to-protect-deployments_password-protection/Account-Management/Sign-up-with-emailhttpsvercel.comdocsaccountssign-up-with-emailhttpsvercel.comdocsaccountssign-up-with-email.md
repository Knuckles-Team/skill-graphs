##  [Sign up with email](https://vercel.com/docs/accounts#sign-up-with-email)[](https://vercel.com/docs/accounts#sign-up-with-email)
To sign up with email:
  1. Enter your email address to receive the six-digit one-time password (OTP)
  2. Enter the OTP to proceed with logging in successfully.


When signing up with your email, no Git provider will be connected by default. See [login methods and connections](https://vercel.com/docs/accounts#login-methods-and-connections) for information on how to connect a Git provider. If no Git provider is connected, you will be asked to verify your account on every login attempt.
