##  [Removing an email address](https://vercel.com/docs/getting-started-with-vercel#removing-an-email-address)[](https://vercel.com/docs/getting-started-with-vercel#removing-an-email-address)
To remove an email address select the Delete button in the dot menu.
If you wish to remove your primary email address, you will need to set a new primary email address first.
