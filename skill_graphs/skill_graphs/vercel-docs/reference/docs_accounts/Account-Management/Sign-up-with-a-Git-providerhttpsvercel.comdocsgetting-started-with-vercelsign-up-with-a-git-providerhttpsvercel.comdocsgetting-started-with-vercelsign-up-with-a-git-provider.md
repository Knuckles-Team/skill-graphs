##  [Sign up with a Git provider](https://vercel.com/docs/getting-started-with-vercel#sign-up-with-a-git-provider)[](https://vercel.com/docs/getting-started-with-vercel#sign-up-with-a-git-provider)
You can sign up with any of the following supported Git providers:
  * [GitHub](https://vercel.com/docs/git/vercel-for-github)
  * [GitLab](https://vercel.com/docs/git/vercel-for-gitlab)
  * [Bitbucket](https://vercel.com/docs/git/vercel-for-bitbucket)


Authorize Vercel to access your Git provider account. This will be the default login connection on your account.
Once signed up you can manage your login connections in the [authentication section](https://vercel.com/account/authentication) of your dashboard.
