##  [Installing Vercel CLI](https://vercel.com/docs/cli#installing-vercel-cli)[](https://vercel.com/docs/cli#installing-vercel-cli)
To download and install Vercel CLI, run the following command:
Terminal
![](https://7nyt0uhk7sse4zvn.public.blob.vercel-storage.com/docs-assets/static/topics/icons/pnpm.svg)pnpm bun yarn npm
```
pnpm i -g vercel
```

```
yarn global add vercel
```

```
npm i -g vercel
```

```
bun add -g vercel
```
