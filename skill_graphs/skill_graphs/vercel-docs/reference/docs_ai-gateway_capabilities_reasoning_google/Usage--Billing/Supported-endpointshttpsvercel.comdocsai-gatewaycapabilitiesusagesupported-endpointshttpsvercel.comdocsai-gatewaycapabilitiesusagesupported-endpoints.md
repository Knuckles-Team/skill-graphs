##  [Supported endpoints](https://vercel.com/docs/ai-gateway/capabilities/usage#supported-endpoints)[](https://vercel.com/docs/ai-gateway/capabilities/usage#supported-endpoints)
You can use the following Usage & Billing endpoints:
  * [`GET /credits`](https://vercel.com/docs/ai-gateway/capabilities/usage#credits) - Check your credit balance and usage information
  * [`GET /generation`](https://vercel.com/docs/ai-gateway/capabilities/usage#generation-lookup) - Retrieve detailed information about a specific generation
