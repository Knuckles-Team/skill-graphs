##  [Working with installations](https://vercel.com/docs#working-with-installations)[](https://vercel.com/docs#working-with-installations)
Use installation endpoints to update status, send notifications, or retrieve installation details:
  * [**Get Installation**](https://vercel.com/get-installation) — Retrieve current installation status and configuration
  * [**Update Installation**](https://vercel.com/update-installation) — Send notifications to users or update installation metadata
  * [**Get Member**](https://vercel.com/get-member) — Retrieve information about team members for access control or audit logs
