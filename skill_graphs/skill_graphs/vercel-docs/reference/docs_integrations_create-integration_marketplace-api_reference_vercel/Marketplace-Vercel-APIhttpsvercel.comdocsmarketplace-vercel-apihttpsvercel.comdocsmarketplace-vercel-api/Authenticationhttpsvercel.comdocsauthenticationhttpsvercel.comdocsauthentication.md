##  [Authentication](https://vercel.com/docs#authentication)[](https://vercel.com/docs#authentication)
Authenticate requests to the Vercel API using a Bearer token. You receive this token during installation.
###  [bearerToken](https://vercel.com/docs#bearertoken)[](https://vercel.com/docs#bearertoken)
This authentication uses the [Vercel API authentication bearer token](https://vercel.com/docs/rest-api#authentication). The access token is provided in the `credentials` field of the request body of the [Upsert Installation](https://vercel.com/docs/integrations/create-integration/marketplace-api/reference/partner/upsert-installation) call.
