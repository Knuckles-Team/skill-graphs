##  [1. Create a Flags Discovery Endpoint](https://vercel.com/docs/flags/vercel-flags/dashboard#1.-create-a-flags-discovery-endpoint)[](https://vercel.com/docs/flags/vercel-flags/dashboard#1.-create-a-flags-discovery-endpoint)
The Flags Explorer reads flag metadata from a well-known API route:
app/.well-known/vercel/flags/route.ts
```
import { createFlagsDiscoveryEndpoint, getProviderData } from 'flags/next';
import * as flags from '../../../../flags';

export const GET = createFlagsDiscoveryEndpoint(async () => {
  return getProviderData(flags);
});
```

This endpoint uses the `FLAGS_SECRET` environment variable to authenticate requests. Make sure you've pulled it with `vercel env pull`.
