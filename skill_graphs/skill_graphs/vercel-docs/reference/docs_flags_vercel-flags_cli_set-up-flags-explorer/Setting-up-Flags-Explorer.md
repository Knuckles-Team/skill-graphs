# Setting up Flags Explorer
Last updated March 8, 2026
The [Flags Explorer](https://vercel.com/docs/flags/flags-explorer) adds a panel to the [Vercel Toolbar](https://vercel.com/docs/vercel-toolbar) that lets you override flag values on preview deployments. Make sure you've [set up the toolbar](https://vercel.com/docs/vercel-toolbar) first. This is a one-time setup per project.
