##  [Extended usage](https://vercel.com/docs/cli/build#extended-usage)[](https://vercel.com/docs/cli/build#extended-usage)
terminal
```
vercel --cwd [path-to-project]
```

Using the `vercel` command and supplying a path to the root directory of the Vercel project.
terminal
```
vercel deploy --prebuilt
```

Using the `vercel` command to deploy a prebuilt Vercel project, typically with `vercel build`. See [vercel build](https://vercel.com/docs/cli/build) and [Build Output API](https://vercel.com/docs/build-output-api/v3) for more details.
