# NEXTJS_USE_NEXT_IMAGE
Last updated March 4, 2025
Conformance is available on [Enterprise plans](https://vercel.com/docs/plans/enterprise)
This rule is available from version 1.1.0.
The Next.js Image component (`<img>` element with features for automatic image optimization.
It optimizes image sizes for different devices using modern image formats, improves visual stability by preventing layout shifts during image loading, and speeds up page loads with lazy loading and optional blur-up placeholders.
Additionally, it provides the flexibility of on-demand image resizing, even for images hosted on remote servers. This may incur costs from your managed hosting provider (see [below](https://vercel.com/docs/getting-started-with-vercel#important-note-on-costs) for more information)
By default, this rule is disabled. Enable it by [customizing Conformance](https://vercel.com/docs/conformance/customize).
