##  [Setting up an XM Cloud project, environment, and website](https://vercel.com/docs/integrations/cms/agility-cms#setting-up-an-xm-cloud-project-environment-and-website)[](https://vercel.com/docs/integrations/cms/agility-cms#setting-up-an-xm-cloud-project-environment-and-website)
  1. ###  [Access XM Cloud Deploy app](https://vercel.com/docs/integrations/cms/agility-cms#access-xm-cloud-deploy-app)[](https://vercel.com/docs/integrations/cms/agility-cms#access-xm-cloud-deploy-app)
Log in to your XM Cloud Deploy app account.
  2. ###  [Initiate project creation](https://vercel.com/docs/integrations/cms/agility-cms#initiate-project-creation)[](https://vercel.com/docs/integrations/cms/agility-cms#initiate-project-creation)
Navigate to the Projects page and select Create project.
![](https://vercel.com/vc-ap-vercel-docs/_next/image?url=https%3A%2F%2F7nyt0uhk7sse4zvn.public.blob.vercel-storage.com%2Fdocs-assets%2Fstatic%2Fdocs%2Fintegrations%2Fsitecore%2Fsitecore-create-project.png&w=3840&q=75)![](https://vercel.com/vc-ap-vercel-docs/_next/image?url=https%3A%2F%2F7nyt0uhk7sse4zvn.public.blob.vercel-storage.com%2Fdocs-assets%2Fstatic%2Fdocs%2Fintegrations%2Fsitecore%2Fsitecore-create-project.png&w=3840&q=75)
  3. ###  [Select project foundation](https://vercel.com/docs/integrations/cms/agility-cms#select-project-foundation)[](https://vercel.com/docs/integrations/cms/agility-cms#select-project-foundation)
In the Create new project dialog, select Start from the XM Cloud starter foundation. Proceed by selecting Next.
![](https://vercel.com/vc-ap-vercel-docs/_next/image?url=https%3A%2F%2F7nyt0uhk7sse4zvn.public.blob.vercel-storage.com%2Fdocs-assets%2Fstatic%2Fdocs%2Fintegrations%2Fsitecore%2Fsitecore-create-project-modal.png&w=3840&q=75)![](https://vercel.com/vc-ap-vercel-docs/_next/image?url=https%3A%2F%2F7nyt0uhk7sse4zvn.public.blob.vercel-storage.com%2Fdocs-assets%2Fstatic%2Fdocs%2Fintegrations%2Fsitecore%2Fsitecore-create-project-modal.png&w=3840&q=75)
  4. ###  [Select starter template](https://vercel.com/docs/integrations/cms/agility-cms#select-starter-template)[](https://vercel.com/docs/integrations/cms/agility-cms#select-starter-template)
Select the XM Cloud Foundation starter template and select Next.
![](https://vercel.com/vc-ap-vercel-docs/_next/image?url=https%3A%2F%2F7nyt0uhk7sse4zvn.public.blob.vercel-storage.com%2Fdocs-assets%2Fstatic%2Fdocs%2Fintegrations%2Fsitecore%2Fsitecore-create-project-modal-next.png&w=3840&q=75)![](https://vercel.com/vc-ap-vercel-docs/_next/image?url=https%3A%2F%2F7nyt0uhk7sse4zvn.public.blob.vercel-storage.com%2Fdocs-assets%2Fstatic%2Fdocs%2Fintegrations%2Fsitecore%2Fsitecore-create-project-modal-next.png&w=3840&q=75)
  5. ###  [Name your project](https://vercel.com/docs/integrations/cms/agility-cms#name-your-project)[](https://vercel.com/docs/integrations/cms/agility-cms#name-your-project)
Provide a name for your project in the Project name field and select Next.
![](https://vercel.com/vc-ap-vercel-docs/_next/image?url=https%3A%2F%2F7nyt0uhk7sse4zvn.public.blob.vercel-storage.com%2Fdocs-assets%2Fstatic%2Fdocs%2Fintegrations%2Fsitecore%2Fsitecore-create-project-modal-name.png&w=3840&q=75)![](https://vercel.com/vc-ap-vercel-docs/_next/image?url=https%3A%2F%2F7nyt0uhk7sse4zvn.public.blob.vercel-storage.com%2Fdocs-assets%2Fstatic%2Fdocs%2Fintegrations%2Fsitecore%2Fsitecore-create-project-modal-name.png&w=3840&q=75)
  6. ###  [Select source control provider](https://vercel.com/docs/integrations/cms/agility-cms#select-source-control-provider)[](https://vercel.com/docs/integrations/cms/agility-cms#select-source-control-provider)
Choose your source control provider and select Next.
![](https://vercel.com/vc-ap-vercel-docs/_next/image?url=https%3A%2F%2F7nyt0uhk7sse4zvn.public.blob.vercel-storage.com%2Fdocs-assets%2Fstatic%2Fdocs%2Fintegrations%2Fsitecore%2Fsitecore-create-project-modal-provider.png&w=3840&q=75)![](https://vercel.com/vc-ap-vercel-docs/_next/image?url=https%3A%2F%2F7nyt0uhk7sse4zvn.public.blob.vercel-storage.com%2Fdocs-assets%2Fstatic%2Fdocs%2Fintegrations%2Fsitecore%2Fsitecore-create-project-modal-provider.png&w=3840&q=75)
  7. ###  [Set up source control connection](https://vercel.com/docs/integrations/cms/agility-cms#set-up-source-control-connection)[](https://vercel.com/docs/integrations/cms/agility-cms#set-up-source-control-connection)
If you haven't already set up a connection, create a new source control connection and follow the instructions provided by your source control provider.
![](https://vercel.com/vc-ap-vercel-docs/_next/image?url=https%3A%2F%2F7nyt0uhk7sse4zvn.public.blob.vercel-storage.com%2Fdocs-assets%2Fstatic%2Fdocs%2Fintegrations%2Fsitecore%2Fsitecore-create-project-modal-connection.png&w=3840&q=75)![](https://vercel.com/vc-ap-vercel-docs/_next/image?url=https%3A%2F%2F7nyt0uhk7sse4zvn.public.blob.vercel-storage.com%2Fdocs-assets%2Fstatic%2Fdocs%2Fintegrations%2Fsitecore%2Fsitecore-create-project-modal-connection.png&w=3840&q=75)
  8. ###  [Specify repository name](https://vercel.com/docs/integrations/cms/agility-cms#specify-repository-name)[](https://vercel.com/docs/integrations/cms/agility-cms#specify-repository-name)
In the Repository name field, provide a unique name for your new repository and select Next.
![](https://vercel.com/vc-ap-vercel-docs/_next/image?url=https%3A%2F%2F7nyt0uhk7sse4zvn.public.blob.vercel-storage.com%2Fdocs-assets%2Fstatic%2Fdocs%2Fintegrations%2Fsitecore%2Fsitecore-create-project-modal-repo.png&w=3840&q=75)![](https://vercel.com/vc-ap-vercel-docs/_next/image?url=https%3A%2F%2F7nyt0uhk7sse4zvn.public.blob.vercel-storage.com%2Fdocs-assets%2Fstatic%2Fdocs%2Fintegrations%2Fsitecore%2Fsitecore-create-project-modal-repo.png&w=3840&q=75)
  9. ###  [Configure environment details](https://vercel.com/docs/integrations/cms/agility-cms#configure-environment-details)[](https://vercel.com/docs/integrations/cms/agility-cms#configure-environment-details)
     * Specify the environment name in the Environment name field
     * Determine if the environment is a production environment using the Production environment drop-down menu
     * Decide if you want automatic deployments upon commits to the linked repository branch using the Trigger deployment on commit to branch drop-down menu
![](https://vercel.com/vc-ap-vercel-docs/_next/image?url=https%3A%2F%2F7nyt0uhk7sse4zvn.public.blob.vercel-storage.com%2Fdocs-assets%2Fstatic%2Fdocs%2Fintegrations%2Fsitecore%2Fsitecore-create-project-modal-env.png&w=3840&q=75)![](https://vercel.com/vc-ap-vercel-docs/_next/image?url=https%3A%2F%2F7nyt0uhk7sse4zvn.public.blob.vercel-storage.com%2Fdocs-assets%2Fstatic%2Fdocs%2Fintegrations%2Fsitecore%2Fsitecore-create-project-modal-env.png&w=3840&q=75)
  10. ###  [Finalize setup](https://vercel.com/docs/integrations/cms/agility-cms#finalize-setup)[](https://vercel.com/docs/integrations/cms/agility-cms#finalize-setup)
Select Create and deploy.
![](https://vercel.com/vc-ap-vercel-docs/_next/image?url=https%3A%2F%2F7nyt0uhk7sse4zvn.public.blob.vercel-storage.com%2Fdocs-assets%2Fstatic%2Fdocs%2Fintegrations%2Fsitecore%2Fsitecore-create-project-deploy.png&w=3840&q=75)![](https://vercel.com/vc-ap-vercel-docs/_next/image?url=https%3A%2F%2F7nyt0uhk7sse4zvn.public.blob.vercel-storage.com%2Fdocs-assets%2Fstatic%2Fdocs%2Fintegrations%2Fsitecore%2Fsitecore-create-project-deploy.png&w=3840&q=75)
  11. ###  [Create a new website](https://vercel.com/docs/integrations/cms/agility-cms#create-a-new-website)[](https://vercel.com/docs/integrations/cms/agility-cms#create-a-new-website)
     * When the deployment finishes, select Go to XM Cloud
![](https://vercel.com/vc-ap-vercel-docs/_next/image?url=https%3A%2F%2F7nyt0uhk7sse4zvn.public.blob.vercel-storage.com%2Fdocs-assets%2Fstatic%2Fdocs%2Fintegrations%2Fsitecore%2Fsitecore-create-project-click.png&w=3840&q=75)![](https://vercel.com/vc-ap-vercel-docs/_next/image?url=https%3A%2F%2F7nyt0uhk7sse4zvn.public.blob.vercel-storage.com%2Fdocs-assets%2Fstatic%2Fdocs%2Fintegrations%2Fsitecore%2Fsitecore-create-project-click.png&w=3840&q=75)
     * Under Sites, select Create Website
![](https://vercel.com/vc-ap-vercel-docs/_next/image?url=https%3A%2F%2F7nyt0uhk7sse4zvn.public.blob.vercel-storage.com%2Fdocs-assets%2Fstatic%2Fdocs%2Fintegrations%2Fsitecore%2Fsitecore-create-website.png&w=3840&q=75)![](https://vercel.com/vc-ap-vercel-docs/_next/image?url=https%3A%2F%2F7nyt0uhk7sse4zvn.public.blob.vercel-storage.com%2Fdocs-assets%2Fstatic%2Fdocs%2Fintegrations%2Fsitecore%2Fsitecore-create-website.png&w=3840&q=75)
     * Select Basic Site
![](https://vercel.com/vc-ap-vercel-docs/_next/image?url=https%3A%2F%2F7nyt0uhk7sse4zvn.public.blob.vercel-storage.com%2Fdocs-assets%2Fstatic%2Fdocs%2Fintegrations%2Fsitecore%2Fsitecore-create-website-basic.png&w=3840&q=75)![](https://vercel.com/vc-ap-vercel-docs/_next/image?url=https%3A%2F%2F7nyt0uhk7sse4zvn.public.blob.vercel-storage.com%2Fdocs-assets%2Fstatic%2Fdocs%2Fintegrations%2Fsitecore%2Fsitecore-create-website-basic.png&w=3840&q=75)
     * Enter a name for your site in the Site name field
     * Select Create website
![](https://vercel.com/vc-ap-vercel-docs/_next/image?url=https%3A%2F%2F7nyt0uhk7sse4zvn.public.blob.vercel-storage.com%2Fdocs-assets%2Fstatic%2Fdocs%2Fintegrations%2Fsitecore%2Fsitecore-create-website-name.png&w=3840&q=75)![](https://vercel.com/vc-ap-vercel-docs/_next/image?url=https%3A%2F%2F7nyt0uhk7sse4zvn.public.blob.vercel-storage.com%2Fdocs-assets%2Fstatic%2Fdocs%2Fintegrations%2Fsitecore%2Fsitecore-create-website-name.png&w=3840&q=75)
  12. ###  [Publish the site](https://vercel.com/docs/integrations/cms/agility-cms#publish-the-site)[](https://vercel.com/docs/integrations/cms/agility-cms#publish-the-site)
     * Select the Open in Pages option on the newly created website
![](https://vercel.com/vc-ap-vercel-docs/_next/image?url=https%3A%2F%2F7nyt0uhk7sse4zvn.public.blob.vercel-storage.com%2Fdocs-assets%2Fstatic%2Fdocs%2Fintegrations%2Fsitecore%2Fsitecore-create-website-open.png&w=3840&q=75)![](https://vercel.com/vc-ap-vercel-docs/_next/image?url=https%3A%2F%2F7nyt0uhk7sse4zvn.public.blob.vercel-storage.com%2Fdocs-assets%2Fstatic%2Fdocs%2Fintegrations%2Fsitecore%2Fsitecore-create-website-open.png&w=3840&q=75)
     * Select Publish > Publish item with all sub-items
![](https://vercel.com/vc-ap-vercel-docs/_next/image?url=https%3A%2F%2F7nyt0uhk7sse4zvn.public.blob.vercel-storage.com%2Fdocs-assets%2Fstatic%2Fdocs%2Fintegrations%2Fsitecore%2Fsitecore-create-website-publish.png&w=3840&q=75)![](https://vercel.com/vc-ap-vercel-docs/_next/image?url=https%3A%2F%2F7nyt0uhk7sse4zvn.public.blob.vercel-storage.com%2Fdocs-assets%2Fstatic%2Fdocs%2Fintegrations%2Fsitecore%2Fsitecore-create-website-publish.png&w=3840&q=75)
