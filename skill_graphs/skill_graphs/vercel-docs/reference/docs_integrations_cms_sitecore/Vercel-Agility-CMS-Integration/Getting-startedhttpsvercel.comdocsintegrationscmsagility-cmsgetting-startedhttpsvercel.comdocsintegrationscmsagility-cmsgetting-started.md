##  [Getting started](https://vercel.com/docs/integrations/cms/agility-cms#getting-started)[](https://vercel.com/docs/integrations/cms/agility-cms#getting-started)
To get started with the Agility CMS on Vercel deploy the template below:
[![Blog with Agility CMS and Next.js](https://vercel.com/vc-ap-vercel-docs/_next/image?url=https%3A%2F%2Fimages.ctfassets.net%2Fe5382hct74si%2F4GSn6MPh4fozgXFvlpc9d7%2Fd4338bcdcdbfde5714fc6dd011fdd75f%2Fagility-cms.png&w=3840&q=75) Blog with Agility CMS and Next.js Start a blog with Next.js, AgilityCMS and TailwindCSS. ](https://vercel.com/templates/next.js/blog-agility-cms-nextjs)
Or, follow the steps below to install the integration:
  1. ###  [Install the Vercel CLI](https://vercel.com/docs/integrations/cms/agility-cms#install-the-vercel-cli)[](https://vercel.com/docs/integrations/cms/agility-cms#install-the-vercel-cli)
To pull in environment variables from Agility CMS to your Vercel project, you need to install the [Vercel CLI](https://vercel.com/docs/cli). Run the following command in your terminal:
Terminal
![](https://7nyt0uhk7sse4zvn.public.blob.vercel-storage.com/docs-assets/static/topics/icons/pnpm.svg)pnpm bun yarn npm
```
pnpm i -g vercel@latest
```

```
yarn global add vercel@latest
```

```
npm i -g vercel@latest
```

```
bun add -g vercel@latest
```

  2. ###  [Install your CMS integration](https://vercel.com/docs/integrations/cms/agility-cms#install-your-cms-integration)[](https://vercel.com/docs/integrations/cms/agility-cms#install-your-cms-integration)
Navigate to the [Agility CMS integration](https://vercel.com/integrations/agility-cms) and follow the steps to install the integration.
  3. ###  [Pull in environment variables](https://vercel.com/docs/integrations/cms/agility-cms#pull-in-environment-variables)[](https://vercel.com/docs/integrations/cms/agility-cms#pull-in-environment-variables)
Once you've installed the Agility CMS integration, you can pull in environment variables from Agility CMS to your Vercel project. In your terminal, run:
```
vercel env pull
```



See your installed CMSs documentation for next steps on how to use the integration.
