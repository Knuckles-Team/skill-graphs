##  [Query parameters](https://vercel.com/docs#query-parameters)[](https://vercel.com/docs#query-parameters)
includeHostedZonesbooleanOptional
Whether to include Hosted Zones in the response
includePeeringConnectionsbooleanOptional
Whether to include VPC Peering connections in the response
includeProjectsbooleanOptional
Whether to include projects in the response
searchstringOptional
The query to use as a filter for returned networks
teamIdstringOptional
The Team identifier to perform the request on behalf of.
slugstringOptional
The Team slug to perform the request on behalf of.
