# Add the Vercel Toolbar to local and production environments
Last updated May 23, 2025
The Vercel Toolbar is available by default on all [preview environments](https://vercel.com/docs/deployments/environments#preview-environment-pre-production). In production environments the toolbar supports ongoing team collaboration and project iteration. When used in development environments, you can see and resolve preview comments during development, streamlining the process of iterating on your project.
All toolbar features such as [Comments](https://vercel.com/docs/comments/using-comments), [Feature Flags](https://vercel.com/docs/feature-flags), [Draft Mode](https://vercel.com/docs/draft-mode), and [Edit Mode](https://vercel.com/docs/edit-mode), are available in both production and development environments.
  * [Add the toolbar to your local or production environment](https://vercel.com/docs/vercel-toolbar/in-production-and-localhost/add-to-localhost)
