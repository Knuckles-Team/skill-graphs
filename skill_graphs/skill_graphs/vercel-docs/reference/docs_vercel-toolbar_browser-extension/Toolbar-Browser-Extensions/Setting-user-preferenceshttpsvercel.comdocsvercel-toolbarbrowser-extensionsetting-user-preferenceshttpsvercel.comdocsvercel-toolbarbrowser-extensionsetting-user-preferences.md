##  [Setting user preferences](https://vercel.com/docs/vercel-toolbar/browser-extension#setting-user-preferences)[](https://vercel.com/docs/vercel-toolbar/browser-extension#setting-user-preferences)
With the browser extension you are able to toggle on the following preferences that affect how the toolbar behaves for you without altering its behavior for your team members:
Setting | Description
---|---
Always Activate | Sets the toolbar to activate anytime you are authenticated as your Vercel user instead of waiting to be clicked.
Start Hidden | Sets the toolbar to start hidden. Read more about [hiding and showing the toolbar](https://vercel.com/docs/vercel-toolbar/managing-toolbar#disable-toolbar-for-session).
