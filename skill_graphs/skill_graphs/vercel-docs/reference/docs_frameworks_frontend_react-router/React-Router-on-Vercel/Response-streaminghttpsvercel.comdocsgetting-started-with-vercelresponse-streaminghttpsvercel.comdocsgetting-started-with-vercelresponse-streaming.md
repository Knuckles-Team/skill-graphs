##  [Response streaming](https://vercel.com/docs/getting-started-with-vercel#response-streaming)[](https://vercel.com/docs/getting-started-with-vercel#response-streaming)
[Streaming HTTP responses](https://vercel.com/docs/functions/streaming-functions)
with React Router on Vercel is supported with Vercel Functions. See the
Streaming with React Router on Vercel:
  * Offers faster Function response times, improving your app's user experience
  * Allows you to return large amounts of data without exceeding Vercel Function response size limits
  * Allows you to display Instant Loading UI from the server with React Router's `<Await>`
