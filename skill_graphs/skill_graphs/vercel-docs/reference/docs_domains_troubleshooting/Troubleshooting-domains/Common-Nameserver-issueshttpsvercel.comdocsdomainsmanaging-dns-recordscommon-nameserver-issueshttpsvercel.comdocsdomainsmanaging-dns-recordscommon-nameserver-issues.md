###  [Configuring nameservers for wildcard domains](https://vercel.com/docs/domains/managing-dns-records#configuring-nameservers-for-wildcard-domains)[](https://vercel.com/docs/domains/managing-dns-records#configuring-nameservers-for-wildcard-domains)
When you add any custom domain to your Vercel project you must [configure](https://vercel.com/docs/domains/add-a-domain#configure-the-domain) the DNS records with your DNS provider so it can be used with your project. When you add a wildcard domain (such as `*.example.com`), you must [use the Nameservers method](https://vercel.com/docs/domains/add-a-domain#vercel-nameservers).
