##  [Get started with FastAPI on Vercel](https://vercel.com/docs/getting-started-with-vercel#get-started-with-fastapi-on-vercel)[](https://vercel.com/docs/getting-started-with-vercel#get-started-with-fastapi-on-vercel)
You can quickly deploy a FastAPI application to Vercel by creating a FastAPI app or using an existing one:
[![](https://api-frameworks.vercel.sh/framework-logos/fastapi.svg)Deploy FastAPI to Vercel](https://vercel.com/templates/python/fastapi-python-boilerplate)
[Deploy](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Fvercel%2Fvercel%2Ftree%2Fmain%2Fexamples%2Ffastapi&template=fastapi)
###  [Get started with Vercel CLI](https://vercel.com/docs/getting-started-with-vercel#get-started-with-vercel-cli)[](https://vercel.com/docs/getting-started-with-vercel#get-started-with-vercel-cli)
Get started by initializing a new FastAPI project using [Vercel CLI init command](https://vercel.com/docs/cli/init):
terminal
```
vc init fastapi
```

This will clone the `fastapi`.
