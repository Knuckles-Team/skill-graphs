##  [Existing Monitoring users](https://vercel.com/docs/observability#existing-monitoring-users)[](https://vercel.com/docs/observability#existing-monitoring-users)
Monitoring is now automatically included with [Observability Plus](https://vercel.com/docs/observability/observability-plus) and cannot be purchased separately. For existing Monitoring users, [the Monitoring section in the sidebar](https://vercel.com/docs/observability/monitoring) on your dashboard will continue to exist and can be used in the same way that you've always used it.
Teams that are currently paying for Monitoring, will not automatically see the [Observability Plus](https://vercel.com/docs/observability/observability-plus) features and benefits on the Observability tab, but will be able to see [reduced pricing](https://vercel.com/changelog/monitoring-pricing-reduced-up-to-87). In order to use [Observability Plus](https://vercel.com/docs/observability/observability-plus) you should [migrate using the modal](https://vercel.com/docs/observability/observability-plus#enabling-observability-plus). Once you upgrade to Observability Plus, you cannot roll back to the original Monitoring plan. To learn more, see [Monitoring Limits and Pricing](https://vercel.com/docs/observability/monitoring/limits-and-pricing).
In addition, teams that subscribe to [Observability Plus](https://vercel.com/docs/observability/observability-plus) will have access to the Monitoring tab and its features.
