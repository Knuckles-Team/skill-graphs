##  [Connecting to the Marketplace](https://vercel.com/docs/getting-started-with-vercel#connecting-to-the-marketplace)[](https://vercel.com/docs/getting-started-with-vercel#connecting-to-the-marketplace)
Vercel enables you to use Postgres by integrating with external database providers. By using the Marketplace, you can:
  * Select from a [range of Postgres providers](https://vercel.com/marketplace?category=storage&search=postgres)
  * Provision and configure a Postgres database with minimal setup.
  * Have credentials and [environment variables](https://vercel.com/docs/environment-variables) injected into your Vercel project.
