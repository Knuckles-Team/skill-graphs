##  [3. Pull environment variables](https://vercel.com/docs/flags/vercel-flags/cli/roll-out-feature#3.-pull-environment-variables)[](https://vercel.com/docs/flags/vercel-flags/cli/roll-out-feature#3.-pull-environment-variables)
The `FLAGS` environment variable contains your SDK keys. Pull it into your local `.env.local`:
terminal
```
vercel env pull
```
