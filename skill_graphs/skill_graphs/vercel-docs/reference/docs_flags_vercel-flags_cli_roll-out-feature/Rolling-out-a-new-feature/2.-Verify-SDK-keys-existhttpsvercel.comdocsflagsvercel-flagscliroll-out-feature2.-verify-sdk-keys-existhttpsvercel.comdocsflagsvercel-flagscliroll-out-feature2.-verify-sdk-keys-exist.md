##  [2. Verify SDK keys exist](https://vercel.com/docs/flags/vercel-flags/cli/roll-out-feature#2.-verify-sdk-keys-exist)[](https://vercel.com/docs/flags/vercel-flags/cli/roll-out-feature#2.-verify-sdk-keys-exist)
Each project gets SDK keys automatically when you create your first flag. Confirm they're in place:
terminal
```
vercel flags sdk-keys ls
```
