##  [Supported ciphers](https://vercel.com/docs/cdn-security/encryption#supported-ciphers)[](https://vercel.com/docs/cdn-security/encryption#supported-ciphers)
To protect data integrity, Vercel only supports strong ciphers with
Supported cipher algorithms:
  * `TLS_AES_128_GCM_SHA256` (TLS 1.3)
  * `TLS_AES_256_GCM_SHA384` (TLS 1.3)
  * `TLS_CHACHA20_POLY1305_SHA256` (TLS 1.3)
  * `ECDHE-ECDSA-AES128-GCM-SHA256` (TLS 1.2)
  * `ECDHE-RSA-AES128-GCM-SHA256` (TLS 1.2)
  * `ECDHE-ECDSA-AES256-GCM-SHA384` (TLS 1.2)
  * `ECDHE-RSA-AES256-GCM-SHA384` (TLS 1.2)
  * `ECDHE-ECDSA-CHACHA20-POLY1305` (TLS 1.2)
  * `ECDHE-RSA-CHACHA20-POLY1305` (TLS 1.2)
  * `DHE-RSA-AES256-GCM-SHA384` (TLS 1.2)
