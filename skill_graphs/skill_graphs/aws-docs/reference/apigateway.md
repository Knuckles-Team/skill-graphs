# Amazon API Gateway Documentation
Amazon API Gateway enables you to create and deploy your own REST and WebSocket APIs at any scale. You can create robust, secure, and scalable APIs that access Amazon Web Services or other web services, as well as data that’s stored in the AWS Cloud. You can create APIs to use in your own client applications, or you can make your APIs available to third-party app developers.
  1. ### [Developer Guide](https://docs.aws.amazon.com/apigateway/latest/developerguide/index.html)
Provides a conceptual overview of Amazon API Gateway and includes detailed instructions for using the service.
     * [HTML](https://docs.aws.amazon.com/apigateway/latest/developerguide/index.html)
     * [PDF](https://docs.aws.amazon.com/apigateway/latest/developerguide/apigateway-dg.pdf)
  2. ### [REST API Reference](https://docs.aws.amazon.com/apigateway/api-reference/)
