# Amazon Simple Storage Service Documentation
Amazon Simple Storage Service (Amazon S3) is storage for the internet. You can use Amazon S3 to store and retrieve any amount of data at any time, from anywhere on the web.
## Amazon S3
Use Amazon S3 to store, manage, analyze, and protect any amount of data for virtually any use case, such as data lakes, cloud-native applications, and mobile apps.
  1. ### [User Guide](https://docs.aws.amazon.com/AmazonS3/latest/userguide/index.html)
Provides detailed information and instructions for getting started, developing, and working with Amazon S3 using the AWS Management Console, AWS CLI, AWS SDKs, and REST API.
     * [HTML](https://docs.aws.amazon.com/AmazonS3/latest/userguide/index.html)
     * [PDF](https://docs.aws.amazon.com/AmazonS3/latest/userguide/s3-userguide.pdf)
  2. ### [API Reference](https://docs.aws.amazon.com/AmazonS3/latest/API/Welcome.html)
Describes all the Amazon S3 API operations in detail. Also provides sample requests, responses, and errors for the supported web services protocols.
     * [HTML](https://docs.aws.amazon.com/AmazonS3/latest/API/Welcome.html)
     * [PDF](https://docs.aws.amazon.com/AmazonS3/latest/API/s3-api.pdf)


##  Amazon Glacier
Use Amazon Glacier for data archiving use cases and the lowest cost archive storage in the cloud.
