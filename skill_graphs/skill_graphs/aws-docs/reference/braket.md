# Amazon Braket Documentation
Amazon Braket is a fully managed service that helps you get started with quantum computing by providing a development environment to explore and design quantum algorithms, test them on simulated quantum computers, and run them on your choice of different quantum hardware technologies.
  1. ### [Developer Guide](https://docs.aws.amazon.com/braket/latest/developerguide/index.html)
Provides a conceptual overview of Amazon Braket, and detailed information about how to design and create quantum tasks, test them on quantum simulators, and then run them on a quantum computer.
     * [HTML](https://docs.aws.amazon.com/braket/latest/developerguide/index.html)
     * [PDF](https://docs.aws.amazon.com/braket/latest/developerguide/braket-developer-guide.pdf)
  2. ### [API Reference](https://docs.aws.amazon.com/braket/latest/APIReference/index.html)
The Amazon Braket API Reference provides information about the operations and structures supported by Amazon Braket.
     * [HTML](https://docs.aws.amazon.com/braket/latest/APIReference/index.html)
     * [PDF](https://docs.aws.amazon.com/braket/latest/APIReference/amazon-braket-api.pdf)
  3. Allows you to interact with Amazon Braket using the AWS SDK for Python (Boto3).
  4. The Amazon Braket GitHub organization provides a good starting point to discover multiple frameworks and educational content. The material is open sourced and builds on top of Amazon Braket.
