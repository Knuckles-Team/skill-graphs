###  [[edit](https://en.cppreference.com/mwiki/index.php?title=c/23&action=edit&section=2 "Edit section: Feature test macros for optional features")] Feature test macros for optional features
  * [`__STDC_IEC_60559_BFP__`](https://en.cppreference.com/w/c/preprocessor/replace.html "c/preprocessor/replace")
    * Indicates IEEE-754 binary floating-point arithmetic and required math functions are supported.       This macro supersedes [`__STDC_IEC_559__`](https://en.cppreference.com/w/c/preprocessor/replace.html "c/preprocessor/replace").
  * [`__STDC_IEC_60559_DFP__`](https://en.cppreference.com/w/c/preprocessor/replace.html "c/preprocessor/replace")
    * Indicates IEEE-754 decimal floating-point arithmetic and required math functions are supported.
  * [`__STDC_IEC_60559_COMPLEX__`](https://en.cppreference.com/w/c/preprocessor/replace.html "c/preprocessor/replace")
    * Indicates IEEE-754 complex arithmetic and required math functions are supported.       This macro supersedes [`__STDC_IEC_559_COMPLEX__`](https://en.cppreference.com/w/c/preprocessor/replace.html "c/preprocessor/replace").

##  [[edit](https://en.cppreference.com/mwiki/index.php?title=c/23&action=edit&section=6 "Edit section: Obsolete features")] Obsolete features
###  [[edit](https://en.cppreference.com/mwiki/index.php?title=c/23&action=edit&section=7 "Edit section: Removed features")] Removed features
  * Old-style [function declarations](https://en.cppreference.com/w/c/language/function_declaration.html "c/language/function declaration") and [function definitions](https://en.cppreference.com/w/c/language/function_definition.html "c/language/function definition")
  * Representations for [signed integers](https://en.cppreference.com/w/c/language/arithmetic_types.html "c/language/arithmetic types") other than two's complement
  * Permission that u/U-prefixed [character constants](https://en.cppreference.com/w/c/language/character_constant.html "c/language/character constant") and [string literals](https://en.cppreference.com/w/c/language/string_literal.html "c/language/string literal") may be not UTF-16/32
  * Mixed wide [string literal](https://en.cppreference.com/w/c/language/string_literal.html "c/language/string literal") concatenation
  * Support for calling [`realloc()`](https://en.cppreference.com/w/c/memory/realloc.html "c/memory/realloc") with zero size (the behavior becomes undefined)
  * [`alignas`](https://en.cppreference.com/w/c/language/alignas.html "c/language/alignas") and [`alignof`](https://en.cppreference.com/w/c/language/alignof.html "c/language/alignof") are provided as macros defined in [`<stdalign.h>`](https://en.cppreference.com/w/c/header/stdalign.html "c/header/stdalign") (becomes keywords)
  * [`static_assert`](https://en.cppreference.com/w/c/error/static_assert.html "c/error/static assert") is provided as a macro defined in [`<assert.h>`](https://en.cppreference.com/w/c/header/assert.html "c/header/assert") (becomes a keyword)
  * [`thread_local`](https://en.cppreference.com/w/c/thread/thread_local.html "c/thread/thread local") is provided as a macro defined in [`<threads.h>`](https://en.cppreference.com/w/c/header/threads.html "c/header/threads") (becomes a keyword)
