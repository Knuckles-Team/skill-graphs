## What's new for C++ in Visual Studio 2022
Visual Studio 2022 brings many updates and fixes to the Microsoft C++ compiler and tools. The Visual Studio IDE also offers significant improvements in performance and productivity, and now runs natively as a 64-bit application.
  * For more information on what's new in all of Visual Studio, see [What's new in Visual Studio 2022](https://learn.microsoft.com/en-us/visualstudio/ide/whats-new-visual-studio-2022).
  * For information about version build dates, see [Visual Studio 2022 Release History](https://learn.microsoft.com/en-us/visualstudio/releases/2022/release-history).
