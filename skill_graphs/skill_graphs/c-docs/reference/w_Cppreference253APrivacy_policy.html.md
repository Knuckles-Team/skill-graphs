# Cppreference:Privacy policy
From cppreference.com
We're not doing anything creepy:
  * If you make anonymous edits to pages, we'll publicly display your IP address next to your changes.
  * If you create an account and log in, we'll bake a tasty cookie and use it to remember you.
  * We use Google analytics to track statistics.
  * We keep recent Apache traffic logs.


In other words, we do everything you would expect from a friendly neighborhood geek-run wiki.
