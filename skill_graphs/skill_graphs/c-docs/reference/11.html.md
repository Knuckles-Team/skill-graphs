##  [[edit](https://en.cppreference.com/mwiki/index.php?title=c/11&action=edit&section=1 "Edit section: New language features")] New language features
  * Multithreaded environments
    * [Atomic objects](https://en.cppreference.com/w/c/language/atomic.html "c/language/atomic") ([`_Atomic`](https://en.cppreference.com/w/c/keyword/_Atomic.html "c/keyword/ Atomic"))
    * [Thread local storage](https://en.cppreference.com/w/c/language/storage_class_specifiers.html#Storage_duration "c/language/storage duration") ([`_Thread_local`](https://en.cppreference.com/w/c/keyword/_Thread_local.html "c/keyword/ Thread local"))
  * Enhanced alignment support
    * [Alignment query](https://en.cppreference.com/w/c/language/alignof.html "c/language/ Alignof") ([`_Alignof`](https://en.cppreference.com/w/c/keyword/_Alignof.html "c/keyword/ Alignof"))
    * [Alignment strengthening](https://en.cppreference.com/w/c/language/alignas.html "c/language/ Alignas") ([`_Alignas`](https://en.cppreference.com/w/c/keyword/_Alignas.html "c/keyword/ Alignas"))
    * Over-aligned types
  * Unicode support
    * [`u`/`U` character constants](https://en.cppreference.com/w/c/language/character_constant.html "c/language/character constant")
    * [`u8`/`u`/`U` string literals](https://en.cppreference.com/w/c/language/string_literal.html "c/language/string literal")
  * [Generic selection expressions](https://en.cppreference.com/w/c/language/generic.html "c/language/generic") ([`_Generic`](https://en.cppreference.com/w/c/keyword/_Generic.html "c/keyword/ Generic"))
  * [Non-returning functions](https://en.cppreference.com/w/c/language/noreturn.html "c/language/ Noreturn") ([`_Noreturn`](https://en.cppreference.com/w/c/keyword/_Noreturn.html "c/keyword/ Noreturn"))
  * Anonymous [struct](https://en.cppreference.com/w/c/language/struct.html "c/language/struct") and [union](https://en.cppreference.com/w/c/language/union.html "c/language/union") members
  * Fine-grained [evaluation order](https://en.cppreference.com/w/c/language/eval_order.html "c/language/eval order")
  * Extending the lifetime of [temporary objects](https://en.cppreference.com/w/c/language/lifetime.html#Temporary_lifetime "c/language/lifetime")
  * [`_Static_assert`](https://en.cppreference.com/w/c/language/static_assert.html "c/language/ Static assert")
  * [Analyzability](https://en.cppreference.com/w/c/language/analyzability.html "c/language/analyzability")


###  [[edit](https://en.cppreference.com/mwiki/index.php?title=c/11&action=edit&section=2 "Edit section: Feature test macros for optional features")] Feature test macros for optional features
  * [`__STDC_ANALYZABLE__`](https://en.cppreference.com/w/c/preprocessor/replace.html "c/preprocessor/replace")
    * Indicates analyzability is supported.
  * [`__STDC_LIB_EXT1__`](https://en.cppreference.com/w/c/preprocessor/replace.html "c/preprocessor/replace")
    * Indicates bounds checking functions are supported.
  * [`__STDC_NO_ATOMICS__`](https://en.cppreference.com/w/c/preprocessor/replace.html "c/preprocessor/replace")
    * Indicates atomic objects and the atomic operation library are not supported.
  * [`__STDC_NO_COMPLEX__`](https://en.cppreference.com/w/c/preprocessor/replace.html "c/preprocessor/replace")
    * Indicates complex types and the complex math functions are not supported.       These features were mandatory in C99.
  * [`__STDC_NO_THREADS__`](https://en.cppreference.com/w/c/preprocessor/replace.html "c/preprocessor/replace")
    * Indicates thread local storage and the thread support library are not supported.
  * [`__STDC_NO_VLA__`](https://en.cppreference.com/w/c/preprocessor/replace.html "c/preprocessor/replace")
    * Indicates [variable length arrays and variably modified types](https://en.cppreference.com/w/c/language/array.html#Variable-length_arrays "c/language/array") are not supported.       These features were mandatory in C99.
