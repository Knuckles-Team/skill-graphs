# [Rust by Example](https://doc.rust-lang.org/rust-by-example/zh/index.html#rust-by-example)
[Rust](https://www.rust-lang.org/) 是一门现代系统编程语言，专注于安全性、速度和并发性。通过内存安全而不使用垃圾回收来实现这些目标。
《通过例子学 Rust》（Rust By Example, RBE）是一系列可运行的示例，它们展示了各种 Rust 概念和标准库。为了更好地利用这些示例，请不要忘记[本地安装 Rust](https://www.rust-lang.org/tools/install)并查看[官方文档](https://doc.rust-lang.org/)。此外，好奇的话，你也可以
现在让我们开始吧！
  * [Hello World](https://doc.rust-lang.org/rust-by-example/zh/hello.html) - 从一个经典的 Hello World 程序开始。
  * [Primitives](https://doc.rust-lang.org/rust-by-example/zh/primitives.html) - 学习有符号整数、无符号整数和其他原生类型。
  * [自定义类型](https://doc.rust-lang.org/rust-by-example/zh/custom_types.html) - 结构体 `struct` 和枚举 `enum`。
  * [变量绑定](https://doc.rust-lang.org/rust-by-example/zh/variable_bindings.html) - 可变绑定、作用域、遮蔽。
  * [类型](https://doc.rust-lang.org/rust-by-example/zh/types.html) - 学习如何改变和定义类型。
  * [转换](https://doc.rust-lang.org/rust-by-example/zh/conversion.html) - 在不同类型的数据之间进行转换，如字符串、整数和浮点数。
  * [表达式](https://doc.rust-lang.org/rust-by-example/zh/expression.html) - 学习表达式及其使用方法。
  * [控制流](https://doc.rust-lang.org/rust-by-example/zh/flow_control.html) - `if`/`else`、`for` 等。
  * [函数](https://doc.rust-lang.org/rust-by-example/zh/fn.html) - 学习方法、闭包和高阶函数。
  * [模块](https://doc.rust-lang.org/rust-by-example/zh/mod.html) - 使用模块组织代码
  * [Crates](https://doc.rust-lang.org/rust-by-example/zh/crates.html) - Crate 是 Rust 中的编译单元。学习如何创建库。
  * [Cargo](https://doc.rust-lang.org/rust-by-example/zh/cargo.html) - 了解官方 Rust 包管理工具的一些基本功能。
  * [属性](https://doc.rust-lang.org/rust-by-example/zh/attribute.html) - 属性是应用于某些模块、crate 或项的元数据。
  * [泛型](https://doc.rust-lang.org/rust-by-example/zh/generics.html) - 学习编写可以适用于多种类型参数的函数或数据类型。
  * [作用域规则](https://doc.rust-lang.org/rust-by-example/zh/scope.html) - 作用域在所有权（ownership）、借用（borrowing）和生命周期（lifetime）中扮演重要角色。
  * [特质](https://doc.rust-lang.org/rust-by-example/zh/trait.html) - 特质（trait）是为未知类型 `Self` 定义的一组方法。
  * [宏](https://doc.rust-lang.org/rust-by-example/zh/macros.html) - 宏是一种编写代码以生成其他代码的方式，也被称为元编程。
  * [错误处理](https://doc.rust-lang.org/rust-by-example/zh/error.html) - 学习 Rust 处理失败的方式。
  * [标准库类型](https://doc.rust-lang.org/rust-by-example/zh/std.html) - 学习 `std` 标准库提供的一些自定义类型。
  * [标准库中的其他内容](https://doc.rust-lang.org/rust-by-example/zh/std_misc.html) - 更多关于文件处理、线程的自定义类型。
  * [测试](https://doc.rust-lang.org/rust-by-example/zh/testing.html) - Rust 中的各种测试方法。
  * [不安全操作](https://doc.rust-lang.org/rust-by-example/zh/unsafe.html) - 学习如何编写和使用不安全代码块。
  * [兼容性](https://doc.rust-lang.org/rust-by-example/zh/compatibility.html) - 应对 Rust 语言的演进及可能出现的兼容性问题。
  * [补充](https://doc.rust-lang.org/rust-by-example/zh/meta.html) - 文档，基准测试。
