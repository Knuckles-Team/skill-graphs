# [Std misc](https://doc.rust-lang.org/rust-by-example/print.html#std-misc)
Many other types are provided by the std library to support things such as:
  * Threads
  * Channels
  * File I/O


These expand beyond what the [primitives](https://doc.rust-lang.org/rust-by-example/print.html#primitives) provide.
