##  [Quick reference](https://vercel.com/docs/ai-gateway/getting-started#quick-reference)[](https://vercel.com/docs/ai-gateway/getting-started#quick-reference)
Option | Type | Description
---|---|---
`order` | `string[]` | Provider slugs in the order they should be attempted
`only` | `string[]` | Restrict routing to only these provider slugs
Both options are set under `providerOptions.gateway` in the AI SDK or `providerOptions` in the REST API. See [Available Providers](https://vercel.com/docs/ai-gateway/models-and-providers/provider-options#available-providers) for the full list of provider slugs.
