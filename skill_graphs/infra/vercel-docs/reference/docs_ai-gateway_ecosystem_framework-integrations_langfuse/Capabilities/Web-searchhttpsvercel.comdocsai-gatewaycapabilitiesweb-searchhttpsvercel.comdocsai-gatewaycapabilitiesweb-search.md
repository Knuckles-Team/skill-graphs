##  [Web search](https://vercel.com/docs/ai-gateway/capabilities#web-search)[](https://vercel.com/docs/ai-gateway/capabilities#web-search)
Enable AI models to search the web during conversations. This capability helps answer questions about current events, recent developments, or any topic requiring up-to-date information.
Two approaches are supported:
  * [Perplexity Search](https://vercel.com/docs/ai-gateway/capabilities/web-search#using-perplexity-search): Add web search to any model, regardless of provider
  * Native provider tools: Use search capabilities built into [Anthropic](https://vercel.com/docs/ai-gateway/capabilities/web-search#anthropic-web-search), [OpenAI](https://vercel.com/docs/ai-gateway/capabilities/web-search#openai-web-search), and [Google](https://vercel.com/docs/ai-gateway/capabilities/web-search#google-web-search) models
