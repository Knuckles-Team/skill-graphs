##  [Secure Backend Access with OIDC Federation](https://vercel.com/docs/project-configuration/vercel-json#secure-backend-access-with-oidc-federation)[](https://vercel.com/docs/project-configuration/vercel-json#secure-backend-access-with-oidc-federation)
This feature allows you to secure access to your backend services by using short-lived, non-persistent tokens that are signed by Vercel's OIDC Identity Provider (IdP).
To learn more, see [Secure Backend Access with OIDC Federation](https://vercel.com/docs/oidc).
