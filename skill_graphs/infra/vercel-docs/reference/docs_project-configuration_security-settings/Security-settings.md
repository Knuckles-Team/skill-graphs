# Security settings
Last updated January 13, 2026
To adjust your project's security settings:
  1. Select your project from your [dashboard](https://vercel.com/dashboard)
  2. Open Settings in the sidebar and select [Security](https://vercel.com/d?to=%2F%5Bteam%5D%2F%5Bproject%5D%2Fsettings%2Fsecurity&title=Go+to+Security+settings)


From here you can enable or disable [Attack Challenge Mode](https://vercel.com/docs/attack-challenge-mode), [Logs and Source Protection](https://vercel.com/docs/project-configuration/vercel-json#build-logs-and-source-protection), [Customer Success Code Visibility](https://vercel.com/docs/project-configuration/vercel-json#customer-success-code-visibility) and [Git Fork Protection](https://vercel.com/docs/project-configuration/vercel-json#git-fork-protection).
