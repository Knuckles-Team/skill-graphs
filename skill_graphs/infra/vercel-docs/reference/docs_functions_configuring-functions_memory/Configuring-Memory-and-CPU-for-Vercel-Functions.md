# Configuring Memory and CPU for Vercel Functions
Last updated September 24, 2025
The memory configuration of a function determines how much memory and CPU a function can use while executing. By default, on Pro and Enterprise, functions execute with 2 GB (1 vCPU) of memory. On Hobby, they will always execute with 2 GB (1 vCPU). You can change the [default memory size for all functions](https://vercel.com/docs/functions#setting-your-default-function-memory-/-cpu-size) in a project.
