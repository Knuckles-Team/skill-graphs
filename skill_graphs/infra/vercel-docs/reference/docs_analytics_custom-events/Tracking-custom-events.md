# Tracking custom events
Last updated September 24, 2025
Custom Events are available on [Enterprise](https://vercel.com/docs/plans/enterprise) and [Pro](https://vercel.com/docs/plans/pro) plans
Vercel Web Analytics allows you to track custom events in your application using the `track()` function. This is useful for tracking user interactions, such as button clicks, form submissions, or purchases.
Make sure you have `@vercel/analytics` version 1.1.0 or later [installed](https://vercel.com/docs/analytics/quickstart#add-@vercel/analytics-to-your-project).
