##  [Limitations](https://vercel.com/docs/notebooks#limitations)[](https://vercel.com/docs/notebooks#limitations)
The following limitations apply to custom data:
  * The number of custom data properties you can pass is limited based on your [plan](https://vercel.com/docs/analytics/limits-and-pricing).
  * Nested objects are not supported.
  * Allowed values are `strings`, `numbers`, `booleans`, and `null`.
  * You cannot set event name, key, or values to longer than 255 characters each.
