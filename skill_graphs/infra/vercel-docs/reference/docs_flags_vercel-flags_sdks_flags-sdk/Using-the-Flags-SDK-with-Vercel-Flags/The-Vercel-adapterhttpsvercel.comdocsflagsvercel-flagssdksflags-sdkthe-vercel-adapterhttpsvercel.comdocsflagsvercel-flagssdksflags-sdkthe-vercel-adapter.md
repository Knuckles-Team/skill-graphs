##  [The Vercel adapter](https://vercel.com/docs/flags/vercel-flags/sdks/flags-sdk#the-vercel-adapter)[](https://vercel.com/docs/flags/vercel-flags/sdks/flags-sdk#the-vercel-adapter)
The `@flags-sdk/vercel` package provides the adapter that connects the Flags SDK to your Vercel Flags project. The [Getting Started guide](https://vercel.com/docs/flags/vercel-flags/quickstart) uses `vercelAdapter()`, which reads the `FLAGS` environment variable automatically and initializes lazily on first evaluation.
If you need to connect to a different [SDK Key](https://vercel.com/docs/flags/vercel-flags/dashboard/sdk-keys) — for example, when working with multiple Vercel Flags projects — use `createVercelAdapter` instead. See [SDK Keys](https://vercel.com/docs/flags/vercel-flags/dashboard/sdk-keys#with-the-flags-sdk) for details.
