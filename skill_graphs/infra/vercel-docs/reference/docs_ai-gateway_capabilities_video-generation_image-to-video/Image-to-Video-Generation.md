# Image-to-Video Generation
Last updated March 8, 2026
Animate a static image into a video. The image you provide becomes the video content itself - you're adding motion to that exact scene.
This is different from [reference-to-video](https://vercel.com/docs/ai-gateway/capabilities/video-generation/reference-to-video), where reference images show the model what characters look like, but the video is a completely new scene.
