##  [3. Start the rolling release](https://vercel.com/docs/rolling-releases/rolling-release-deployment#3.-start-the-rolling-release)[](https://vercel.com/docs/rolling-releases/rolling-release-deployment#3.-start-the-rolling-release)
Begin the rolling release to start shifting traffic to the new deployment:
terminal
```
vercel rolling-release start --dpl <deployment-url>
```

This starts at the first stage (10% of traffic in the example configuration above).
