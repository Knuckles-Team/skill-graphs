##  [O](https://vercel.com/docs/glossary#o)[](https://vercel.com/docs/glossary#o)
###  [Observability](https://vercel.com/docs/glossary#observability)[](https://vercel.com/docs/glossary#observability)
Tools and features that help you monitor, analyze, and understand your application's performance, traffic, and behavior in production.
###  [OIDC (OpenID Connect)](https://vercel.com/docs/glossary#oidc-openid-connect)[](https://vercel.com/docs/glossary#oidc-openid-connect)
A federation protocol that issues short-lived, non-persistent tokens for secure backend access without storing long-lived credentials.
###  [Origin Server](https://vercel.com/docs/glossary#origin-server)[](https://vercel.com/docs/glossary#origin-server)
The server that stores and runs the original version of your application code, where requests are processed when not served from cache.
###  [Output Directory](https://vercel.com/docs/glossary#output-directory)[](https://vercel.com/docs/glossary#output-directory)
The folder containing your final build output after the build process completes, such as `dist`, `build`, or `.next`.
