###  [Trusted IPs](https://vercel.com/docs/glossary#trusted-ips)[](https://vercel.com/docs/glossary#trusted-ips)

###  [Turborepo](https://vercel.com/docs/glossary#turborepo)[](https://vercel.com/docs/glossary#turborepo)
A high-performance build system for monorepos that provides fast incremental builds and remote caching capabilities.
