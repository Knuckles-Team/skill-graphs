##  [Using Vercel's features with Nitro](https://vercel.com/docs/getting-started-with-vercel#using-vercel's-features-with-nitro)[](https://vercel.com/docs/getting-started-with-vercel#using-vercel's-features-with-nitro)
When you deploy a Nitro app to Vercel, you can use Vercel specific features such as [Incremental Static Regeneration (ISR)](https://vercel.com/docs/getting-started-with-vercel#incremental-static-regeneration-isr), [preview deployments](https://vercel.com/docs/deployments/environments#preview-environment-pre-production), [Fluid compute](https://vercel.com/docs/fluid-compute), [Observability](https://vercel.com/docs/getting-started-with-vercel#observability), and [Vercel firewall](https://vercel.com/docs/vercel-firewall) with zero or minimum configuration.
