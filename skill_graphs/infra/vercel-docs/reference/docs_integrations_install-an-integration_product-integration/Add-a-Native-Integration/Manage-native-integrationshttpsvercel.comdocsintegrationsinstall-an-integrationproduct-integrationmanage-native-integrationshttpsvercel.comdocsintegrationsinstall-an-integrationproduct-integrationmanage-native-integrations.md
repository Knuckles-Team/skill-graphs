##  [Manage native integrations](https://vercel.com/docs/integrations/install-an-integration/product-integration#manage-native-integrations)[](https://vercel.com/docs/integrations/install-an-integration/product-integration#manage-native-integrations)
Once installed, you can manage the following aspect of the native integration:
  * View the installed resources (instances of products) and then manage each resource.
  * Connect project(s) to a provisioned resource. For products supporting Log Drains, you can enable them and configure which log sources to forward and the sampling rate.
  * View the invoices and usage for each of your provisioned resources in that installation. See [Billing](https://vercel.com/docs/integrations/create-integration/billing) for details on invoice lifecycle, pricing structures, and refunds.
  * [Uninstall the integration](https://vercel.com/docs/integrations/install-an-integration/product-integration#uninstall-an-integration)


###  [Manage products](https://vercel.com/docs/integrations/install-an-integration/product-integration#manage-products)[](https://vercel.com/docs/integrations/install-an-integration/product-integration#manage-products)
To manage products inside the installed integration:
  1. From your Vercel [dashboard](https://vercel.com/dashboard), open [Integrations](https://vercel.com/d?to=%2F%5Bteam%5D%2F%7E%2Fintegrations&title=Go+to+Integrations) in the sidebar.
  2. Next to the integration, select the Manage button. Native integrations appear with a `billable` badge.
  3. On the Integrations page, under Installed Products, select the card for the product you would like to update to be taken to the product's detail page.


####  [Projects](https://vercel.com/docs/integrations/install-an-integration/product-integration#projects)[](https://vercel.com/docs/integrations/install-an-integration/product-integration#projects)
By selecting the Projects link on the left navigation, you can:
  * Connect a project to the product
  * View a list of existing connections and manage them


####  [Settings](https://vercel.com/docs/integrations/install-an-integration/product-integration#settings)[](https://vercel.com/docs/integrations/install-an-integration/product-integration#settings)
By selecting the Settings link on the left navigation, you can update the following:
  * Product name
  * Manage funds: if you selected a prepaid plan for the product, you can Add funds and manage auto recharge settings
  * Delete the product


####  [Getting Started](https://vercel.com/docs/integrations/install-an-integration/product-integration#getting-started)[](https://vercel.com/docs/integrations/install-an-integration/product-integration#getting-started)
By selecting the Getting Started link on the left navigation, you can view quick steps with sample code on how to use the product in your project.
####  [Usage](https://vercel.com/docs/integrations/install-an-integration/product-integration#usage)[](https://vercel.com/docs/integrations/install-an-integration/product-integration#usage)
By selecting the Usage link on the left navigation, you can view a graph of the funds used over time by this product in all the projects where it was installed.
####  [Resources](https://vercel.com/docs/integrations/install-an-integration/product-integration#resources)[](https://vercel.com/docs/integrations/install-an-integration/product-integration#resources)
Under Resources on the left navigation, you can view a list of links which vary depending on the provider for support, guides and additional resources to help you use the product.
###  [Add more products](https://vercel.com/docs/integrations/install-an-integration/product-integration#add-more-products)[](https://vercel.com/docs/integrations/install-an-integration/product-integration#add-more-products)
To add more products to this integration:
  1. From your Vercel [dashboard](https://vercel.com/dashboard), open [Integrations](https://vercel.com/d?to=%2F%5Bteam%5D%2F%7E%2Fintegrations&title=Go+to+Integrations) in the sidebar.
  2. Next to the integration, select the Manage button. Native integrations appear with a `billable` badge.
  3. On the Integrations page, under More Products, select the Install button for any additional products in that integration that you want to use.


###  [Uninstall an integration](https://vercel.com/docs/integrations/install-an-integration/product-integration#uninstall-an-integration)[](https://vercel.com/docs/integrations/install-an-integration/product-integration#uninstall-an-integration)
Uninstalling an integration automatically removes all associated products and their data.
  1. From your Vercel [dashboard](https://vercel.com/dashboard), open [Integrations](https://vercel.com/d?to=%2F%5Bteam%5D%2F%7E%2Fintegrations&title=Go+to+Integrations) in the sidebar.
  2. Next to the integration, select the Manage button.
  3. At the bottom of the integrations page, under Uninstall, select Uninstall Integration and follow the steps to uninstall.
