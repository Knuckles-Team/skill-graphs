##  [Base URL](https://vercel.com/docs#base-url)[](https://vercel.com/docs#base-url)
Configure your integration's base URL when you create your integration in the [Integration Console](https://vercel.com/d?to=%2F%5Bteam%5D%2F%7E%2Fintegrations%2Fconsole&title=Go+to+Integrations+Console). All Partner API endpoints are relative to this base URL.
`https://your-integration-server.com/api`
