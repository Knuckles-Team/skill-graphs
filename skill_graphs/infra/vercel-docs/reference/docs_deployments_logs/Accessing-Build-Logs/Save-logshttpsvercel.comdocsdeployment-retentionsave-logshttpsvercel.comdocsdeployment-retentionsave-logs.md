##  [Save logs](https://vercel.com/docs/deployment-retention#save-logs)[](https://vercel.com/docs/deployment-retention#save-logs)
You can use [Drains](https://vercel.com/docs/drains) to export, store, and analyze your build logs. Log Drains configuration can be accessed through the Vercel dashboard or through one of our [Logging integrations](https://vercel.com/integrations#logging).
