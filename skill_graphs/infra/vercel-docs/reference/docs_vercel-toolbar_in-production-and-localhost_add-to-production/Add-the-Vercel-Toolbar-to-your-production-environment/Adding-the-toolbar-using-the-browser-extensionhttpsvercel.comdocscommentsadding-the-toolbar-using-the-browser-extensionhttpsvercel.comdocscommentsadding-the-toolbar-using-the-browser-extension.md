##  [Adding the toolbar using the browser extension](https://vercel.com/docs/comments#adding-the-toolbar-using-the-browser-extension)[](https://vercel.com/docs/comments#adding-the-toolbar-using-the-browser-extension)
For team members that use supported browsers and want the most straightforward experience, we recommend using the [Vercel Browser Extension](https://vercel.com/docs/vercel-toolbar/browser-extension) to get access to the toolbar on your team's production sites.
For team members that use browsers for which a Vercel extension is not available, to allow toolbar access for everyone that accesses your site, or if you have more complex rules for when it shows in production, you'll need to [add the `@vercel/toolbar` package](https://vercel.com/docs/vercel-toolbar/in-production-and-localhost/add-to-production#adding-the-toolbar-using-the-@vercel/toolbar-package) to your project.
