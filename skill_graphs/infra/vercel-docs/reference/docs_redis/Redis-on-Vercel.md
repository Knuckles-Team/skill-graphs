# Redis on Vercel
Last updated January 13, 2026
Vercel lets you connect external Redis databases through the [Marketplace](https://vercel.com/marketplace), allowing you to integrate high-performance caching and real-time data storage into your Vercel projects without managing Redis servers.
Vercel KV is no longer available. If you had an existing Vercel KV store, we automatically moved it to [Upstash Redis](https://vercel.com/marketplace/upstash) in December 2024. For new projects, install a [Redis integration from the Marketplace](https://vercel.com/marketplace?category=storage&search=redis).
  * Explore [Marketplace storage redis integrations](https://vercel.com/marketplace?category=storage&search=redis).
  * Learn how to [add a Marketplace native integration](https://vercel.com/docs/integrations/install-an-integration/product-integration).
