##  [Why use Marketplace storage](https://vercel.com/docs/marketplace-storage#why-use-marketplace-storage)[](https://vercel.com/docs/marketplace-storage#why-use-marketplace-storage)
When you install a storage integration from the Marketplace, you get:
  * Simplified provisioning: Create databases without leaving the Vercel dashboard
  * Automatic configuration: Vercel injects connection strings and credentials as [environment variables](https://vercel.com/docs/environment-variables)
  * Unified billing: Pay for storage resources through your Vercel account
