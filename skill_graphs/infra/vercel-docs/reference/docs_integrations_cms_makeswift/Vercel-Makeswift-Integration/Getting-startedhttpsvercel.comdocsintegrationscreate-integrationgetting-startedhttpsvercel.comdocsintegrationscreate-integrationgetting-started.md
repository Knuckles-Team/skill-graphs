##  [Getting started](https://vercel.com/docs/integrations/create-integration#getting-started)[](https://vercel.com/docs/integrations/create-integration#getting-started)
To get started with the Makeswift on Vercel deploy the template below:
[![Makeswift Next.js Starter](https://vercel.com/vc-ap-vercel-docs/_next/image?url=https%3A%2F%2Fimages.ctfassets.net%2Fe5382hct74si%2F1UJXkxJYZ7R3SWG8XThNHy%2F8a531778bb3ac3737612ffe523bdd340%2Fmakeswift-template-thumbnail-1.png&w=3840&q=75) Makeswift Next.js Starter A Next.js application configured out of the box to work with the Makeswift builder. ](https://vercel.com/templates/next.js/makeswift-starter)
Or, follow the steps below to install the integration:
  1. ###  [Install the Vercel CLI](https://vercel.com/docs/integrations/create-integration#install-the-vercel-cli)[](https://vercel.com/docs/integrations/create-integration#install-the-vercel-cli)
To pull in environment variables from Makeswift to your Vercel project, you need to install the [Vercel CLI](https://vercel.com/docs/cli). Run the following command in your terminal:
Terminal
![](https://7nyt0uhk7sse4zvn.public.blob.vercel-storage.com/docs-assets/static/topics/icons/pnpm.svg)pnpm bun yarn npm
```
pnpm i -g vercel@latest
```

```
yarn global add vercel@latest
```

```
npm i -g vercel@latest
```

```
bun add -g vercel@latest
```

  2. ###  [Install your CMS integration](https://vercel.com/docs/integrations/create-integration#install-your-cms-integration)[](https://vercel.com/docs/integrations/create-integration#install-your-cms-integration)
Navigate to the [Makeswift integration](https://vercel.com/integrations/makeswift) and follow the steps to install the integration.
  3. ###  [Pull in environment variables](https://vercel.com/docs/integrations/create-integration#pull-in-environment-variables)[](https://vercel.com/docs/integrations/create-integration#pull-in-environment-variables)
Once you've installed the Makeswift integration, you can pull in environment variables from Makeswift to your Vercel project. In your terminal, run:
```
vercel env pull
```



See your installed CMSs documentation for next steps on how to use the integration.
