# Vercel WAF
Last updated February 9, 2026
Vercel WAF is available on [all plans](https://vercel.com/docs/plans)
Those with the [member](https://vercel.com/docs/rbac/access-roles#member-role), [viewer](https://vercel.com/docs/rbac/access-roles#viewer-role), [developer](https://vercel.com/docs/rbac/access-roles#developer-role) and [administrator](https://vercel.com/docs/rbac/access-roles#project-administrators) roles can access this feature
The Vercel WAF, part of the [Firewall](https://vercel.com/docs/vercel-firewall), provides security controls to [monitor](https://vercel.com/docs/vercel-firewall/firewall-observability#traffic) and [control](https://vercel.com/docs/vercel-firewall/firewall-observability#traffic) the internet traffic to your site through logging, blocking and challenging. When you apply a configuration change to the firewall, it takes effect globally within 300ms and can be instantly [rolled back](https://vercel.com/docs/vercel-firewall/vercel-waf#instant-rollback) to prior configurations.
  * [Configure your first Custom Rule](https://vercel.com/docs/security/vercel-waf/custom-rules)
  * [Add IP Blocks](https://vercel.com/docs/security/vercel-waf/ip-blocking)
  * [Explore WAF Managed Rulesets](https://vercel.com/docs/security/vercel-waf/managed-rulesets)
