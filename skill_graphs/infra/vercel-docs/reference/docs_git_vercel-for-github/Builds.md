# Builds
Last updated January 21, 2026
Turbo build machines are now enabled by default for new Pro projects - [Learn more](https://vercel.com/docs/builds/managing-builds#larger-build-machines)
Vercel automatically performs a build every time you deploy your code, whether you're pushing to a Git repository, importing a project via the dashboard, or using the [Vercel CLI](https://vercel.com/docs/cli). This process compiles, bundles, and optimizes your application so it's ready to serve to your users.
