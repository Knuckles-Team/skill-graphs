##  [Ignored files and folders](https://vercel.com/docs/builds#ignored-files-and-folders)[](https://vercel.com/docs/builds#ignored-files-and-folders)
Some files (e.g., large datasets or personal configuration) might not be needed in your deployment:
  * Vercel automatically ignores certain files (like `.git`) for performance and security.
  * You can read more about how to specify [ignored files and folders](https://vercel.com/docs/builds/build-features#ignored-files-and-folders).
