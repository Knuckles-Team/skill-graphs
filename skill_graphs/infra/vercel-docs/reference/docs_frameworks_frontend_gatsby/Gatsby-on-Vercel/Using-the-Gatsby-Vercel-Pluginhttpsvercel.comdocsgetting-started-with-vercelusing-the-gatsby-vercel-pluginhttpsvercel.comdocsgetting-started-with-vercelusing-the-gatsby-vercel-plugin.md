##  [Using the Gatsby Vercel Plugin](https://vercel.com/docs/getting-started-with-vercel#using-the-gatsby-vercel-plugin)[](https://vercel.com/docs/getting-started-with-vercel#using-the-gatsby-vercel-plugin)
automatically detect Gatsby usage and install the `@vercel/gatsby-plugin-vercel-builder` plugin.
To deploy your Gatsby site to Vercel, do not install the `@vercel/gatsby-plugin-vercel-builder` plugin yourself, or add it to your `gatsby-config.js` file.
Vercel persists your Gatsby project's `.cache` directory across builds.
