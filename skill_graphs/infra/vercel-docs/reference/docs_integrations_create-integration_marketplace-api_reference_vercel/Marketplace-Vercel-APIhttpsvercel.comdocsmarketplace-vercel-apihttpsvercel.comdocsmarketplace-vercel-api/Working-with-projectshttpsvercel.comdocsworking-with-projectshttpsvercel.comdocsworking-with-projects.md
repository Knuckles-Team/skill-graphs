##  [Working with projects](https://vercel.com/docs#working-with-projects)[](https://vercel.com/docs#working-with-projects)
Access project data to understand user configuration and manage resources:
  * [**Get Projects**](https://vercel.com/get-projects) — List all projects in an installation
  * [**Get Project**](https://vercel.com/get-project) — Retrieve detailed project configuration
  * [**Get Environment Variables**](https://vercel.com/get-environment-variables) — Read current environment variable configuration
