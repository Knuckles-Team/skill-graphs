##  [3. Use the toolbar](https://vercel.com/docs/flags/vercel-flags/dashboard#3.-use-the-toolbar)[](https://vercel.com/docs/flags/vercel-flags/dashboard#3.-use-the-toolbar)
Visit the preview URL. The Flags Explorer panel appears in the Vercel Toolbar. Toggle any flag to override its value for your session without affecting other users.
See [Flags Explorer](https://vercel.com/docs/flags/flags-explorer/getting-started) for the full setup guide, including how to share overrides with teammates via URL.
