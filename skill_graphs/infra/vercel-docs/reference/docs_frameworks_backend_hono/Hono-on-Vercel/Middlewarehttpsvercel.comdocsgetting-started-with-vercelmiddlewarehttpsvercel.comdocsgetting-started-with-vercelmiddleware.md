###  [Vercel Routing Middleware](https://vercel.com/docs/getting-started-with-vercel#vercel-routing-middleware)[](https://vercel.com/docs/getting-started-with-vercel#vercel-routing-middleware)
In Vercel, [Routing Middleware](https://vercel.com/docs/routing-middleware) executes code before a request is processed by the application. This gives you a way to handle rewrites, redirects, headers, and more, before returning a response. See [the Routing Middleware documentation](https://vercel.com/docs/routing-middleware) for examples.
