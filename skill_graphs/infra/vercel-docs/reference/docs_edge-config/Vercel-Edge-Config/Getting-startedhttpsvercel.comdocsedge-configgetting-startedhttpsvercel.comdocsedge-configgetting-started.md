##  [Getting started](https://vercel.com/docs/edge-config#getting-started)[](https://vercel.com/docs/edge-config#getting-started)
You can create and manage your Edge Config from either [Vercel REST API](https://vercel.com/docs/edge-config/vercel-api) or [Dashboard](https://vercel.com/docs/edge-config/edge-config-dashboard). You can scope your Edge Configs to your Hobby team or [team](https://vercel.com/docs/accounts/create-a-team), and connect them to as many projects as you want.
To get started, see [our quickstart](https://vercel.com/docs/edge-config/get-started).
