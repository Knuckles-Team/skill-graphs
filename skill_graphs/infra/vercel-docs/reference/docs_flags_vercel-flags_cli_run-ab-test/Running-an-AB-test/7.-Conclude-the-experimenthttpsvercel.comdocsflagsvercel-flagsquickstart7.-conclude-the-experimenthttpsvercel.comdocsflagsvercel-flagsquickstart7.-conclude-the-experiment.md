##  [7. Conclude the experiment](https://vercel.com/docs/flags/vercel-flags/quickstart#7.-conclude-the-experiment)[](https://vercel.com/docs/flags/vercel-flags/quickstart#7.-conclude-the-experiment)
When you've picked a winner, clean up:
  1. Remove the flag from code and keep only the winning layout
  2. Archive the flag:


terminal
```
vercel flags archive new-pricing-layout --yes
```
