##  [2. Define the flag in code](https://vercel.com/docs/flags/vercel-flags/quickstart#2.-define-the-flag-in-code)[](https://vercel.com/docs/flags/vercel-flags/quickstart#2.-define-the-flag-in-code)
flags.ts
```
import { flag } from 'flags/next';
import { vercelAdapter } from '@flags-sdk/vercel';

export const newPricingLayout = flag({
  key: 'new-pricing-layout',
  adapter: vercelAdapter(),
});
```

The flag returns `false` until you configure targeting in the dashboard.
