##  [Limitations](https://vercel.com/docs/multi-tenant#limitations)[](https://vercel.com/docs/multi-tenant#limitations)
  * If your app uses manual OpenTelemetry SDK configuration without the usage of `@vercel/otel`, you will not be able to use [Session Tracing](https://vercel.com/docs/tracing/session-tracing) or [Trace Drains](https://vercel.com/docs/drains/reference/traces).
