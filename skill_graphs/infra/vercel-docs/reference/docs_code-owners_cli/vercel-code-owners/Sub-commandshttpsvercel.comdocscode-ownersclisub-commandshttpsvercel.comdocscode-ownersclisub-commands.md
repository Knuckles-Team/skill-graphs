##  [Sub-commands](https://vercel.com/docs/code-owners/cli#sub-commands)[](https://vercel.com/docs/code-owners/cli#sub-commands)
The following sub-commands are available for this CLI.
###  [`init`](https://vercel.com/docs/code-owners/cli#init)[](https://vercel.com/docs/code-owners/cli#init)
The `init` command sets up code owners files in the repository. See [Getting Started](https://vercel.com/docs/code-owners/getting-started#initalizing-code-owners) for more information on using this command.
###  [`validate`](https://vercel.com/docs/code-owners/cli#validate)[](https://vercel.com/docs/code-owners/cli#validate)
The `validate` command checks the syntax for all Code Owners files in the repository for errors.
