##  [Installing an integration](https://vercel.com/docs/integrations/install-an-integration#installing-an-integration)[](https://vercel.com/docs/integrations/install-an-integration#installing-an-integration)
Using Vercel doesn't stop at the products and features that we provide. Through integrations, you can use third-party platforms or services to extend the capabilities of Vercel by:
  * Connecting your Vercel account and project with a third-party service. See [Add a connectable account](https://vercel.com/docs/integrations/install-an-integration/add-a-connectable-account) to learn more.
  * Buying or subscribing to a product with a third-party service that you will use with your Vercel project. See [Add a Native Integration](https://vercel.com/docs/integrations/install-an-integration/product-integration) to learn more.
  * Installing and provisioning resources from the command line. See [`vercel integration add`](https://vercel.com/docs/cli/integration#vercel-integration-add) to learn more.
  * Interacting with your installed integrations through a chat interface. See [Agent Tools](https://vercel.com/docs/integrations/install-an-integration/agent-tools) to learn more.
