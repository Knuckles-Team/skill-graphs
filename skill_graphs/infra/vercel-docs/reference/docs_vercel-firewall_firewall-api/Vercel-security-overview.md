# Vercel security overview
Last updated April 11, 2025
Cloud-deployed web applications face constant security threats, with attackers launching millions of malicious attacks weekly. Your application, users, and business require robust security measures to stay protected.
A comprehensive security strategy requires active protection, robust policies, and compliance frameworks:
  * [Security governance and policies](https://vercel.com/docs/security#governance-and-policies) ensure long-term organizational safety, maintain regulatory adherence, and establish consistent security practices across teams.
  * A [Multi-layered protection](https://vercel.com/docs/security#multi-layered-protection) system provides active security against immediate threats and attacks.
