##  [Compute defaults](https://vercel.com/docs/regions#compute-defaults)[](https://vercel.com/docs/regions#compute-defaults)
  * Vercel Functions default to running in the `iad1` (Washington, D.C., USA) region. Learn more about [changing function regions](https://vercel.com/docs/functions/regions)


Functions should be executed in the same region as your database, or as close to it as possible, [for the lowest latency](https://vercel.com/docs/functions/configuring-functions/region).
