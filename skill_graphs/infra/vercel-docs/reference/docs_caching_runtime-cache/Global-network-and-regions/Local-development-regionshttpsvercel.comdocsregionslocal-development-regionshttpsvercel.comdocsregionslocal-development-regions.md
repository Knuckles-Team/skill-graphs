##  [Local development regions](https://vercel.com/docs/regions#local-development-regions)[](https://vercel.com/docs/regions#local-development-regions)
When you use [the `vercel dev` CLI command to mimic your deployment environment locally](https://vercel.com/docs/cli/dev), the region is assigned `dev1` to mimic the Vercel platform infrastructure.
Region Code | Reference Location
---|---
dev1 | localhost
