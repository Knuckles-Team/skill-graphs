##  [5. Remove conditionals from components](https://vercel.com/docs/flags/vercel-flags#5.-remove-conditionals-from-components)[](https://vercel.com/docs/flags/vercel-flags#5.-remove-conditionals-from-components)
Keep only the code path that was behind the enabled flag:
Before
```
const show = await oldOnboardingFlow();
return show ? <NewOnboarding /> : <OldOnboarding />;
```

After
```
return <NewOnboarding />;
```

Delete any component files that are no longer referenced.
