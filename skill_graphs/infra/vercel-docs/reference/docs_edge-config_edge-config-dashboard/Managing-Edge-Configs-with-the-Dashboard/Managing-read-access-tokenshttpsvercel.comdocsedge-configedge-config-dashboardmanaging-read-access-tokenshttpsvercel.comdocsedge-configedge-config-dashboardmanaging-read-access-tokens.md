List of tokens
To delete a token, click on the vertical ellipsis Delete Token and confirm by clicking Delete Token in the dialog.
You can copy the connection string to be used in your code by clicking on Copy Connection String from the same pop up from the vertical ellipsis
You can generate a new token by clicking the Generate Token button, typing a name slug in the dialog that opens and clicking Create.
