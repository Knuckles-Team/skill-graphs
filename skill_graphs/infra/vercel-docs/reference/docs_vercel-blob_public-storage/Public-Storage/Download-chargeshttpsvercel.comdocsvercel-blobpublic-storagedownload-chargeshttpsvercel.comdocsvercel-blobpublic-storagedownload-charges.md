##  [Download charges](https://vercel.com/docs/vercel-blob/public-storage#download-charges)[](https://vercel.com/docs/vercel-blob/public-storage#download-charges)
When a public blob is downloaded, you pay for:
  1. Browser fetches the blob: [Blob Data Transfer](https://vercel.com/docs/vercel-blob#blob-data-transfer) + [Fast Origin Transfer](https://vercel.com/docs/pricing/networking#fast-origin-transfer) on cache miss


Blob Data Transfer is 3x more cost-efficient than [Fast Data Transfer](https://vercel.com/docs/cdn) on average, making public storage ideal for large media files like images, videos, and documents.
See [pricing documentation](https://vercel.com/docs/vercel-blob/usage-and-pricing) for full details.
