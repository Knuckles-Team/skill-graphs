Choose from multiple APIs that provide compatibility with popular NoSQL and relational database models for modern applications.

### MongoDB
Migrate existing MongoDB applications seamlessly with full wire protocol compatibility and familiar MongoDB tools.
  * [Supported features and syntax](https://learn.microsoft.com/en-us/azure/cosmos-db/mongodb/feature-support-42)
  * [Mapping consistency levels](https://learn.microsoft.com/en-us/azure/cosmos-db/mongodb/consistency-mapping)

Build highly scalable applications using the Cassandra Query Language (CQL) with automatic global distribution.

  * [Partitioning](https://learn.microsoft.com/en-us/azure/cosmos-db/cassandra/partitioning)

  * [Wire protocol support](https://learn.microsoft.com/en-us/azure/cosmos-db/gremlin/support)
