# AWS SDK for SAP ABAP
AWS SDK for SAP ABAP provides an interface to Amazon Web Services in ABAP. Using the SDK, you can build SAP applications on top of Amazon S3, Amazon EC2, DynamoDB, and more.
  1. ### [AWS SDK for SAP ABAP Developer Guide](https://docs.aws.amazon.com/sdk-for-sapabap/latest/developer-guide/home.html)
Introduces AWS SDK for SAP ABAP and provides information about using the SDK, including code examples for a number of services.
     * [HTML](https://docs.aws.amazon.com/sdk-for-sapabap/latest/developer-guide/home.html)
     * [PDF](https://docs.aws.amazon.com/sdk-for-sapabap/latest/developer-guide/AWS%20SDK%20for%20SAP%20ABAP.pdf)
  2. ### [API Reference](https://docs.aws.amazon.com/sdk-for-sap-abap/v1/api/latest/index.html)
Describes the API operations of AWS SDK for SAP ABAP in detail. The guide also provides sample requests, responses, and errors for the supported web services protocols.
     * [HTML](https://docs.aws.amazon.com/sdk-for-sap-abap/v1/api/latest/index.html)
  3. ### [Code Library](https://docs.aws.amazon.com/code-library/latest/ug/index.html)
A collection of code examples that show you how to use AWS SDKs with AWS. It is organized by service and by SDK to help you find the examples you need.
     * [HTML](https://docs.aws.amazon.com/code-library/latest/ug/index.html)
     * [PDF](https://docs.aws.amazon.com/code-library/latest/ug/code-example-library.pdf)
