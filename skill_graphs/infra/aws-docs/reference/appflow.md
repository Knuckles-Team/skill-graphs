# Amazon AppFlow Documentation
Amazon AppFlow is a fully managed API integration service that you use to connect your software as a service (SaaS) applications to AWS services, and securely transfer data. Use Amazon AppFlow flows to manage and automate your data transfers without needing to write code.
  1. ### [User Guide](https://docs.aws.amazon.com/appflow/latest/userguide/index.html)
Describes key concepts and provides instructions for using Amazon AppFlow.
     * [HTML](https://docs.aws.amazon.com/appflow/latest/userguide/index.html)
     * [PDF](https://docs.aws.amazon.com/appflow/latest/userguide/appflow.pdf)
  2. ### [API Reference](https://docs.aws.amazon.com/appflow/1.0/APIReference/index.html)
Describes all the API operations for Amazon AppFlow in detail. Also provides sample requests, responses, and errors for the supported web service protocols.
     * [HTML](https://docs.aws.amazon.com/appflow/1.0/APIReference/index.html)
     * [PDF](https://docs.aws.amazon.com/appflow/1.0/APIReference/amazon-appflow-api.pdf)
