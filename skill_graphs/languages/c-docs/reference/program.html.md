###  [[edit](https://en.cppreference.com/mwiki/index.php?title=c/program&action=edit&section=1 "Edit section: Program termination")] Program termination
The following functions manage program termination and resources cleanup.
Defined in header `[`<stdlib.h>`](https://en.cppreference.com/w/c/header/stdlib.html "c/header/stdlib")`
---
[ abort](https://en.cppreference.com/w/c/program/abort.html "c/program/abort") |  causes abnormal program termination (without cleaning up)
(function) [[edit]](https://en.cppreference.com/mwiki/index.php?title=Template:c/program/dsc_abort&action=edit)
[ exit](https://en.cppreference.com/w/c/program/exit.html "c/program/exit") |  causes normal program termination with cleaning up
(function) [[edit]](https://en.cppreference.com/mwiki/index.php?title=Template:c/program/dsc_exit&action=edit)
[ quick_exit](https://en.cppreference.com/w/c/program/quick_exit.html "c/program/quick exit") (C11) |  causes normal program termination without completely cleaning up
(function) [[edit]](https://en.cppreference.com/mwiki/index.php?title=Template:c/program/dsc_quick_exit&action=edit)
[ _Exit](https://en.cppreference.com/w/c/program/_Exit.html "c/program/ Exit") (C99) |  causes normal program termination without cleaning up
(function) [[edit]](https://en.cppreference.com/mwiki/index.php?title=Template:c/program/dsc_Exit&action=edit)
[ atexit](https://en.cppreference.com/w/c/program/atexit.html "c/program/atexit") |  registers a function to be called on [exit()](https://en.cppreference.com/w/c/program/exit.html "c/program/exit") invocation
(function) [[edit]](https://en.cppreference.com/mwiki/index.php?title=Template:c/program/dsc_atexit&action=edit)
[ at_quick_exit](https://en.cppreference.com/w/c/program/at_quick_exit.html "c/program/at quick exit") (C11) |  registers a function to be called on [`quick_exit`](https://en.cppreference.com/w/c/program/quick_exit.html "c/program/quick exit") invocation
(function) [[edit]](https://en.cppreference.com/mwiki/index.php?title=Template:c/program/dsc_at_quick_exit&action=edit)
[ EXIT_SUCCESSEXIT_FAILURE](https://en.cppreference.com/w/c/program/EXIT_status.html "c/program/EXIT status") |  indicates program execution execution status
(macro constant) [[edit]](https://en.cppreference.com/mwiki/index.php?title=Template:c/program/dsc_EXIT_status&action=edit)

###  [[edit](https://en.cppreference.com/mwiki/index.php?title=c/program&action=edit&section=5 "Edit section: Signals")] Signals
Several functions and macro constants for signal management are provided.
Defined in header `[`<signal.h>`](https://en.cppreference.com/w/c/header/signal.html "c/header/signal")`
---
[ signal](https://en.cppreference.com/w/c/program/signal.html "c/program/signal") |  sets a signal handler for particular signal
(function) [[edit]](https://en.cppreference.com/mwiki/index.php?title=Template:c/program/dsc_signal&action=edit)
[ raise](https://en.cppreference.com/w/c/program/raise.html "c/program/raise") |  runs the signal handler for particular signal
(function) [[edit]](https://en.cppreference.com/mwiki/index.php?title=Template:c/program/dsc_raise&action=edit)
[ sig_atomic_t](https://en.cppreference.com/w/c/program/sig_atomic_t.html "c/program/sig atomic t") |  the integer type that can be accessed as an atomic entity from an asynchronous signal handler
(typedef)
[ SIG_DFLSIG_IGN](https://en.cppreference.com/w/c/program/SIG_strategies.html "c/program/SIG strategies") |  defines signal handling strategies
(macro constant) [[edit]](https://en.cppreference.com/mwiki/index.php?title=Template:c/program/dsc_SIG_strategies&action=edit)
[ SIG_ERR](https://en.cppreference.com/w/c/program/SIG_ERR.html "c/program/SIG ERR") |  error was encountered
(macro constant) [[edit]](https://en.cppreference.com/mwiki/index.php?title=Template:c/program/dsc_SIG_ERR&action=edit)
