## What's new for C++ in Visual Studio version 16.11
For a summary of new features and bug fixes in Visual Studio version 16.11, see [What's New in Visual Studio 2019 version 16.11](https://learn.microsoft.com/en-us/visualstudio/releases/2019/release-notes-v16.11).
  * The compiler now supports the [`/std:c++20`](https://learn.microsoft.com/en-us/cpp/build/reference/std-specify-language-standard-version?view=msvc-170) compiler mode. Previously, C++20 features were available only in [`/std:c++latest`](https://learn.microsoft.com/en-us/cpp/build/reference/std-specify-language-standard-version?view=msvc-170) mode in Visual Studio 2019. Features that originally required **`/std:c++latest`**mode now work in**`/std:c++20`**mode or later in the latest versions of Visual Studio.
