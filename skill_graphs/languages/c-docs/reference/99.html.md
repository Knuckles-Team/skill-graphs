##  [[edit](https://en.cppreference.com/mwiki/index.php?title=c/99&action=edit&section=1 "Edit section: New language features")] New language features
  * Universal-character-names in [identifiers](https://en.cppreference.com/w/c/language/identifiers.html "c/language/identifier")
  * Increased [translation limits](https://en.cppreference.com/w/c/language/identifiers.html#Translation_limits "c/language/identifier")
  * // [comments](https://en.cppreference.com/w/c/comment.html "c/comment")
  * [`restrict`](https://en.cppreference.com/w/c/language/restrict.html "c/language/restrict") pointers
  * Enhanced [arithmetic types](https://en.cppreference.com/w/c/language/arithmetic_types.html "c/language/arithmetic types")
    * _Bool
    * long long and unsigned long long
    * Extended integer types
    * [`_Complex`](https://en.cppreference.com/w/c/keyword/_Complex.html "c/keyword/ Complex") types (float _Complex, double _Complex, and long double _Complex)
    * [`_Imaginary`](https://en.cppreference.com/w/c/keyword/_Imaginary.html "c/keyword/ Imaginary") types (float _Imaginary, double _Imaginary, and long double _Imaginary)
  * Flexible array members
  * [Variable-length array](https://en.cppreference.com/w/c/language/array.html#Variable-length_arrays "c/language/array") (VLA) types and variably-modified (VM) types
  * Improvements of braced-init-list for [array](https://en.cppreference.com/w/c/language/array_initialization.html "c/language/array initialization"), [struct and union](https://en.cppreference.com/w/c/language/struct_initialization.html "c/language/struct initialization") types
    * Non-constant initializers
    * Designated initializers
  * Idempotent cvr-qualifiers
  * Trailing comma in [enumerator-list](https://en.cppreference.com/w/c/language/enum.html "c/language/enum")
  * Hexadecimal [floating constants](https://en.cppreference.com/w/c/language/floating_constant.html "c/language/floating constant")
  * [Compound literals](https://en.cppreference.com/w/c/language/compound_literal.html "c/language/compound literal")
  * Floating-point environment
  * Requiring truncation for divisions of signed integer types
  * Implicit return 0; in the [`main()` function](https://en.cppreference.com/w/c/language/main_function.html "c/language/main function")
  * Declarations and statements in mixed order
  * init-statement in [`for`](https://en.cppreference.com/w/c/language/for.html "c/language/for") loops
  * [`inline`](https://en.cppreference.com/w/c/language/inline.html "c/language/inline") functions
  * Predefined variable [`__func__`](https://en.cppreference.com/w/c/language/function_definition.html "c/language/function definition")
  * Cvr-qualifiers and static in [] within function declarations
  * [Variadic macros](https://en.cppreference.com/w/c/preprocessor/replace.html "c/preprocessor/replace")
  * [`_Pragma`](https://en.cppreference.com/w/c/preprocessor/impl.html "c/preprocessor/impl") preprocessor operator
  * Standard pragmas for floating-point evaluation
    * `STDC` [`FENV_ACCESS`](https://en.cppreference.com/w/c/preprocessor/impl.html "c/preprocessor/impl")
    * `STDC` [`FP_CONTRACT`](https://en.cppreference.com/w/c/preprocessor/impl.html "c/preprocessor/impl")
    * `STDC` [`CX_LIMITED_RANGE`](https://en.cppreference.com/w/c/preprocessor/impl.html "c/preprocessor/impl")


###  [[edit](https://en.cppreference.com/mwiki/index.php?title=c/99&action=edit&section=2 "Edit section: Feature test macros for optional features")] Feature test macros for optional features
  * [`__STDC_IEC_559__`](https://en.cppreference.com/w/c/preprocessor/replace.html "c/preprocessor/replace")
    * Indicates IEEE-754 binary floating-point arithmetic and required math functions are supported.
  * [`__STDC_IEC_559_COMPLEX__`](https://en.cppreference.com/w/c/preprocessor/replace.html "c/preprocessor/replace")
    * Indicates IEEE-754 complex arithmetic and required math functions are supported.
  * [`__STDC_HOSTED__`](https://en.cppreference.com/w/c/preprocessor/replace.html "c/preprocessor/replace")
    * Indicates that the implementation is [hosted](https://en.cppreference.com/w/c/language/conformance.html "c/language/conformance").
  * [`__STDC_ISO_10646__`](https://en.cppreference.com/w/c/preprocessor/replace.html "c/preprocessor/replace")
    * Indicates that Unicode is used by the wide literal encodings and expands to the latest supported revision.
  * [`__STDC_MB_MIGHT_NEQ_WC__`](https://en.cppreference.com/w/c/preprocessor/replace.html "c/preprocessor/replace")
    * Indicates that there are some characters in the basic character set having different code unit values in [ordinary and wide literal encodings](https://en.cppreference.com/w/c/language/character_constant.html "c/language/character constant").

###  [[edit](https://en.cppreference.com/mwiki/index.php?title=c/99&action=edit&section=5 "Edit section: Library features")] Library features
  * [Aliases for integer types](https://en.cppreference.com/w/c/types/integer.html "c/types/integer")
    * Integer types with exact width (`int_N__t`and`uint _N__t`)
    * Fastest integer types with at least given width (`int_fast_N__t`and`uint_fast _N__t`)
    * Smallest integer types with at least given width (`int_least_N__t`and`uint_least _N__t`)
    * Integer types capable for cast between object pointers ([intptr_t](https://en.cppreference.com/w/c/types/integer.html "c/types/integer") and [uintptr_t](https://en.cppreference.com/w/c/types/integer.html "c/types/integer"))
    * Integer types with maximum width ([intmax_t](https://en.cppreference.com/w/c/types/integer.html "c/types/integer") and [uintmax_t](https://en.cppreference.com/w/c/types/integer.html "c/types/integer"))
  * Operations on long long and [intmax_t](https://en.cppreference.com/w/c/types/integer.html "c/types/integer")
    * [llabs()](https://en.cppreference.com/w/c/numeric/math/abs.html "c/numeric/math/abs")
    * [imaxabs()](https://en.cppreference.com/w/c/numeric/math/abs.html "c/numeric/math/abs")
    * [lldiv()](https://en.cppreference.com/w/c/numeric/math/div.html "c/numeric/math/div")
    * [imaxdiv()](https://en.cppreference.com/w/c/numeric/math/div.html "c/numeric/math/div")
  * [Floating-point environment access](https://en.cppreference.com/w/c/numeric/fenv.html "c/numeric/fenv")
  * Extended floating-point math functions
    * New floating-point math functions
    * -`f` and -`l` variants for existing and new floating-point math functions
    * [Math error handling](https://en.cppreference.com/w/c/numeric/math/math_errhandling.html "c/numeric/math/math errhandling")
  * [Complex functions](https://en.cppreference.com/w/c/numeric/complex.html "c/numeric/complex")
  * [_Exit()](https://en.cppreference.com/w/c/program/_Exit.html "c/program/ Exit")
  * Formatting support for long long, unsigned long long, [intmax_t](https://en.cppreference.com/w/c/types/integer.html "c/types/integer"), and [uintmax_t](https://en.cppreference.com/w/c/types/integer.html "c/types/integer")
    * [atoll()](https://en.cppreference.com/w/c/string/byte/atoi.html "c/string/byte/atoi")
    * [strtoimax()](https://en.cppreference.com/w/c/string/byte/strtoimax.html "c/string/byte/strtoimax")
    * [strtoll()](https://en.cppreference.com/w/c/string/byte/strtol.html "c/string/byte/strtol")
    * [strtoull](https://en.cppreference.com/w/c/string/byte/strtoul.html "c/string/byte/strtoul")
    * [strtoumax()](https://en.cppreference.com/w/c/string/byte/strtoimax.html "c/string/byte/strtoimax")
    * [wcstoimax()](https://en.cppreference.com/w/c/string/wide/wcstoimax.html "c/string/wide/wcstoimax")
    * [wcstoll()](https://en.cppreference.com/w/c/string/wide/wcstol.html "c/string/wide/wcstol")
    * [wcstoull()](https://en.cppreference.com/w/c/string/wide/wcstoul.html "c/string/wide/wcstoul")
    * [wcstoumax()](https://en.cppreference.com/w/c/string/wide/wcstoimax.html "c/string/wide/wcstoimax")
  * [isblank()](https://en.cppreference.com/w/c/string/byte/isblank.html "c/string/byte/isblank") and [iswblank()](https://en.cppreference.com/w/c/string/wide/iswblank.html "c/string/wide/iswblank")
  * [snprintf()](https://en.cppreference.com/w/c/io/fprintf.html "c/io/fprintf") and [vsnprintf()](https://en.cppreference.com/w/c/io/vfprintf.html "c/io/vfprintf")
  * [vfscanf()](https://en.cppreference.com/w/c/io/vfscanf.html "c/io/vfscanf") and [vfwscanf()](https://en.cppreference.com/w/c/io/vfwscanf.html "c/io/vfwscanf") function families
  * Extensions for [`fscanf()`](https://en.cppreference.com/w/c/io/fscanf.html "c/io/fscanf") and [`fprintf()`](https://en.cppreference.com/w/c/io/fprintf.html "c/io/fprintf") function families
    * `ll` length modifier for long long and unsigned long long
    * `hh` length modifier for signed char and unsigned char
    * `l` length modifier for double
    * `z` length modifier for [size_t](https://en.cppreference.com/w/c/types/size_t.html "c/types/size t") and its signed version
    * `t` length modifier for [ptrdiff_t](https://en.cppreference.com/w/c/types/ptrdiff_t.html "c/types/ptrdiff t") and its unsigned version
    * `j` length modifier for [intmax_t](https://en.cppreference.com/w/c/types/integer.html "c/types/integer") and [uintmax_t](https://en.cppreference.com/w/c/types/integer.html "c/types/integer")
    * `a` conversion specifier for floating-point types
  * Numeric limit macros for long long and unsigned long long
  * Numeric limit macros corresponding to existing and new aliases for integer types
  * [Format string macros for integer types](https://en.cppreference.com/w/c/types/integer.html#Format_macro_constants "c/types/integer")
  * [`va_copy`](https://en.cppreference.com/w/c/variadic/va_copy.html "c/variadic/va copy")
  * [Type-generic math macros](https://en.cppreference.com/w/c/numeric/tgmath.html "c/numeric/tgmath")
  * Floating-point comparison macros
  * Floating-point classification macros
  * Compatibility macros for _Bool ([`bool`](https://en.cppreference.com/w/c/types.html "c/types"), [`true`](https://en.cppreference.com/w/c/types.html "c/types"), and [`false`](https://en.cppreference.com/w/c/types.html "c/types"))
