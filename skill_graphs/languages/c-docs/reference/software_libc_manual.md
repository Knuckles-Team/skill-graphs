### The GNU C Library (glibc) manual
* * *
This glibc manual version 2.43 (latest) is available in the following formats:
  * [HTML ](https://sourceware.org/glibc/manual/latest/html_mono/libc.html) - entirely on one web page.
  * [HTML](https://sourceware.org/glibc/manual/latest/html_node/index.html) - one web page per node.
  * [HTML compressed (gzipped tar file)](https://sourceware.org/glibc/manual/latest/html_node/libc-html_node.tar.gz) - with one web page per node.
  * [Info document (gzipped tar file)](https://sourceware.org/glibc/manual/latest/info/libc-info.tar.gz).
  * [ASCII text compressed (gzipped)](https://sourceware.org/glibc/manual/latest/text/libc.txt.gz).
  * [TeX dvi file (gzipped)](https://sourceware.org/glibc/manual/latest/dvi/libc.dvi.gz).
  * [PDF file](https://sourceware.org/glibc/manual/latest/pdf/libc.pdf).
  * [Texinfo source (gzipped tar file)](https://sourceware.org/glibc/manual/latest/texi/libc-texi.tar.gz).
