หน้าต่าง ๆ ใน ArchWiki จะถูกเขียนตามข้อกำหนดของเรา ถ้าคุณอ่านแล้วไม่เข้าใจบางส่วน ลองดูคำอธิบายที่ [Help:Reading](https://wiki.archlinux.org/title/Help:Reading "Help:Reading") เผื่อจะเข้าใจได้ดีขึ้น
ถ้าคุณต้องการค้นหาข้อมูล ลองใช้กล่องค้นหาที่ด้านซ้ายของหน้า หรือใช้เครื่องมือค้นหาในเว็บและกรองผลการค้นหาสำหรับวิกินี้ก็ได้ (เช่น เพิ่มตัวกรอง `site:wiki.archlinux.org` ต่อท้ายจากคำที่ต้องการค้นหา) สำหรับเกร็ดการค้นหาข้อมูล ลองดูที่ [Help:Searching](https://wiki.archlinux.org/title/Help:Searching "Help:Searching")
เครื่องมือ captcha ที่ใช้เพื่อลงทะเบียนบัญชี wiki ใหม่บังคับให้คุณต้องมีระบบ Arch ที่อัพเดตแล้ว อย่างไรก็ตาม เรายินดีต้อนรับคนที่ใช้ระบบปฏิบัติการอื่นด้วยเช่นกัน โดยมีข้อแม้ว่าคุณมีส่วนร่วมในเนื้อหาที่เกี่ยวกับลีนุกซ์ [โดยทั่วไปเท่านั้น](https://wiki.archlinux.org/title/ArchWiki:Contributing#Non-Arch_users "ArchWiki:Contributing"): สำหรับคำตอบของ captcha เราแนะนำให้คุณบู๊ตเครื่องจาก [live image](https://archlinux.org/download/) ล่าสุดของ Arch ใน virtual machine หรือไม่ก็ส่งอีเมล์ถึง [ผู้ดูแลระบบ](https://wiki.archlinux.org/title/ArchWiki:Administrators "ArchWiki:Administrators") เพื่อขอความช่วยเหลือในด้านนี้แทน
