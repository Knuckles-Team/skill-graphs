**jikes**

Fast Java compiler adhering to language and VM specifications Jikes is a Java compiler that translates Java source files as defined in The Java Language Specification (Addison-Wesley, 1996) into the bytecoded instruction set and binary format defined in The Java Virtual Machine Specification (Addison-Wesley, 1996). Unlike other compilers, Jikes accepts the Java language only as specified: not as a subset, variant, or superset. In addition to strictly adhering to specifications, Jikes is faster than most compilers and can compute the complete dependency relations in program files. This allows for the generation of dependency makefiles suitable for use with make. Jikes can also be run in incremental mode: after the initial compilation, Jikes waits until prompted, determines which files have been changed, and then does the minimal amount of work needed to bring the class files into a complete and consistent state. This cycle can be repeated until the q command is entered to end the compilation. From Debian 3.0r0 APT <http://www.tldp.org/LDP/Linux-Dictionary/html/index.html>
