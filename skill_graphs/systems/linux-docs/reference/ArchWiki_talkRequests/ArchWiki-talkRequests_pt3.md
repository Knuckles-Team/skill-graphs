**Notice for editors**
Here, list [redirects](https://wiki.archlinux.org/title/Help:Editing#Redirects "Help:Editing") of ambiguous value, that **do not have a history** (we [do not](https://wiki.archlinux.org/title/Help:Procedures#Remove_an_entire_page "Help:Procedures") delete pages with a history) outside of changing the redirect target. Such redirects may have been created when [moving a page](https://wiki.archlinux.org/title/Special:MovePage "Special:MovePage") which had a [bad title](https://wiki.archlinux.org/title/Help:Article_naming_guidelines "Help:Article naming guidelines") initially, or before
Before placing a request here, we encourage you to:
  * use the [Special:WhatLinksHere](https://wiki.archlinux.org/title/Special:WhatLinksHere "Special:WhatLinksHere") page from the offending redirect—as an example see [[1]](https://wiki.archlinux.org/title/Special:WhatLinksHere/ArchWiki:Requests),
  * adjust all the page that still point to it, i.e. edit all such pages and replace redirect in question with correct content.


As you see, there is no good reason to remove a redirect, if some page still uses it.
