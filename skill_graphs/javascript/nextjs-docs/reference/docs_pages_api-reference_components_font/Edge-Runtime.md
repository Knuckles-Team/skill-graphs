# Edge Runtime
Last updated February 27, 2026
Next.js has two server runtimes you can use in your application:
  * The **Node.js Runtime** (default), which has access to all Node.js APIs and is used for rendering your application.
  * The **Edge Runtime** which contains a more limited [set of APIs](https://nextjs.org/docs/pages/api-reference/edge#reference), used in [Proxy](https://nextjs.org/docs/app/api-reference/file-conventions/proxy).
