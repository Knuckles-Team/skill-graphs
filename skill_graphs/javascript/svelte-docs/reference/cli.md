The command line interface (CLI), `sv`, is a toolkit for creating and maintaining Svelte applications.
##  Usage[](https://svelte.dev/docs/cli/overview#Usage)
The easiest way to run `sv` is with `pnpm dlx` if you're using
```
npx sv <command> <args>
```

If you're inside a project where `sv` is already installed, this will use the local installation, otherwise it will download the latest version and run it without installing it, which is particularly useful for [`sv create`](https://svelte.dev/docs/cli/sv-create).
