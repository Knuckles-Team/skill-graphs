# Tooltip
[Previous](https://ui.shadcn.com/docs/components/radix/toggle-group)[Next](https://ui.shadcn.com/docs/components/radix/typography)
A popup that displays information related to an element when the element receives keyboard focus or the mouse hovers over it.
[Radix UI](https://ui.shadcn.com/docs/components/radix/tooltip)[Base UI](https://ui.shadcn.com/docs/components/base/tooltip)

### Add the `TooltipProvider` to the root of your app.
```
Copyimport { TooltipProvider } from "@/components/ui/tooltip"

export default function RootLayout({ children }) {
  return (
    <html lang="en">
      <body>
        <TooltipProvider>{children}</TooltipProvider>
      </body>
    </html>
  )
}
```

## Usage
```
Copyimport {
  Tooltip,
  TooltipContent,
  TooltipTrigger,
} from "@/components/ui/tooltip"
```
```
Copy<Tooltip>
  <TooltipTrigger>Hover</TooltipTrigger>
  <TooltipContent>
    <p>Add to library</p>
  </TooltipContent>
</Tooltip>
```
