# Find Available Slot
Copy page
Get the next available time slot for posting to a specific channel.
Copy page
GET
https://api.postiz.com/public/v1 https://{your-domain}/api/public/v1
/
find-slot
/
{id}
Try it
Find next available slot
cURL
```
curl --request GET \
  --url https://api.postiz.com/public/v1/find-slot/{id} \
  --header 'Authorization: <api-key>'
```

200
```
{
  "date": "2025-01-15T10:00:00.000Z"
}
```
